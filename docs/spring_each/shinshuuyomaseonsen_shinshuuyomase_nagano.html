信州よませ温泉（しんしゅうよませ）

所在地：長野県

効能単純温泉

「信州百名山」の一つ、高社山中腹に位置し、善光寺平と北アルプスを望むスキー場の麓に広がる温泉地。朝は信濃川のきらめき、夕方は北信五岳に沈む夕陽 、夜はダイヤをちりばめたような夜景など、時々刻々と変化する眺望を楽しめる。 さらに、眼下には果樹を中心とした田園地帯が広がり、その向こうには遙か善光寺平を一望。「信州サンセットビューポイント百選」のひとつにも選ばれている絶景が出迎えてくれる。 初夏から秋にかけてさくらんぼやブルーベリー、巨峰、りんごなどの果実を味わうことができるのも信州ならでは。 北志賀高原・高社山の中腹、よませ温泉スキー場に湧く温泉。開湯は平成6年と歴史は浅いが、アルカリ性単純泉の効能豊かな湯は湯治にも良いと評判。毎分800Lと豊富な湯量を誇り、日帰り天然温泉がある。ホテル・旅館街の一角にある「遠見の湯」は、善光寺平から北信濃の山々、千曲川までを一望できる開放感たっぷりの露天風呂。
