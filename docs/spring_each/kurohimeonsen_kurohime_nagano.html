黒姫温泉（くろひめ）

所在地：長野県

泉質炭酸水素塩泉、塩化物泉
効能神経痛、筋肉痛、関節痛、疲労、婦人病、冷え性、皮膚病、痔疾など

黒姫山（くろひめやま）は、長野県上水内郡信濃町にある標高約2053mの成層火山。南東方向より見た整った姿から郷土富士として「信濃富士」とも呼ばれている。 東山麓は黒姫高原として観光地化しており、黒姫高原スノーパークがその面積の大半を占めるが、旬の花々を楽しめることでも有名だ。例年5月中旬からは菜の花、7月中旬からは雪アジサイ、7月下旬からはダリア、9月上旬からはコスモス、そして10月上旬からは紅葉と四季折々の美しい自然に包まれる。 黒姫温泉は黒姫高原スノーパークのそばに位置する温泉で、その広大な国有林に湧出する鉱泉を引湯している。黒姫駅からは車で約10分、信濃町ICからも約10分とアクセスも良好。周辺には野尻湖や牧場もあるので行動範囲が広がりそうだ。 大浴場の大きな窓からは眼下に黒姫高原、正面には日本百名山の妙高山を見ることができる。
