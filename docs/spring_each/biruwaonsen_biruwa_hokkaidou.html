美留和温泉（びるわ）

所在地：北海道

効能ラジウム泉

摩周湖の伏流水が湧き出る森林の中に佇む温泉地、美留和温泉。最寄駅と同じ「びるわおんせん」と読む温泉は、同じく駅に近い別荘地の宿泊施設「きらの宿すばる」だけがややぬるめの源泉から直接引いている。全5室のこぢんまりとした宿は、別料金で宿泊者以外も日帰り入浴ができる24時間天然温泉かけ流しの手作り露天風呂と手作り料理が自慢。屈斜路湖と摩周湖の中間にあたる一帯は静かな環境に囲まれており、体はもちろん心を癒やしに訪れるのにもよさそう。 単純温泉（弱アルカリ性低張性高温泉）の泉質で、神経痛、筋肉痛、関節痛、うちみ、くじき、痔疾、慢性消化器病、冷え性などに効果があるといわれている。源泉は熱いが風呂に入るときはややぬるくなり、程よい湯温で浸かっていられる。
