口坂本温泉（くちさかもと）

所在地：静岡県

効能炭酸水素塩泉

静岡市内とは言っても、山奥にある秘湯が「口坂本温泉」だ。安部川上流の「オクシズ」エリアに属し、大井川上流の井川ダムに至る峠の途中にある（2018年現在通行止め）。5軒の民宿・旅館と、1軒の市営温泉があるのみで、知る人ぞ知る小さな温泉地だ。それぞれの宿が個性を持っていて、山菜や川魚などの山の幸を味わうことができる。いずれの施設も比較的安く訪れることができ、新緑や紅葉などの四季折々の自然を眺めながらのドライブやハイキングを楽しんだ後にゆったりと疲れを癒してくれそうだ。2月には「口坂本宝積寺数珠回し」という長い歴史を持つ祭りも行われる。 泉質は炭酸水素塩泉。pH8.9近くというアルカリ性もあいまったとろみのある独特な触感で、肌触りもよく肌の角質を落として美肌にしてくれる。温度は35℃前後と低いため、加温して提供されている。湧出量は多くはないが、かけ流しも一部でされており、お湯へのこだわりも感じられる。
