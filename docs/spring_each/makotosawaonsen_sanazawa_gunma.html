真沢温泉（さなざわ）

所在地：群馬県

効能単純温泉

月夜野温泉の一部で、自然豊かな大峰山麓に佇む一軒宿「真沢の森」で入れる湯、真沢温泉。「さなざわ」と読む。開放的な大浴場や、星空と自然の風景を楽しめる露天風呂など入るだけでホッとできるような入浴施設の湯は、全国でもトップクラスの高いph値を持つ「美人の湯」。施設内の食堂では、日本随一とされる摘み草料理専門家の監修により、月夜野の山野草をふんだんに使って作る「摘み草料理」を提供しており、食からも薬効が期待できる。内と外で美を手に入れられるほか、周囲にはゴルフ場やスキー場といったレジャー施設のほか、月夜野びーどろパークやたくみの里といった、体験型施設も充実している。 国内トップクラスと言われる高いph値を持つ、メタケイ酸含有、高アルカリ性の泉質で、美肌効果が高いとされる。わずかに白色の浮遊物があり、入って間もなく肌がつるつるして、手ざわりが滑らかになるのが分かるほど効果が高い。無色透明の湯は、肌触りがなめらかでどんな人にも入りやすい。
