天城・持越温泉（あまぎ・もちこし）

所在地：静岡県

効能硫酸塩泉

”湯の国”と呼ばれる天城湯ケ島にある、全く異なる源泉から生まれる7つの温泉・天城七湯のうちの1つ。中伊豆にある自然豊かな温泉地で、狩野川上流に位置しており、ゴルフ場やキャンプ場が点在している。1934～1950年ごろまでは金山として栄えていたが、現在は閉山されている。現在では、釣りやゴルフ、テニス、キャンプなどといったアウトドアが楽しめることもあって、家族連れでにぎわう温泉地だ。神経痛、筋肉痛、関節痛、うちみなどに効果がある。満天の星空と川のせせらぎに耳をすませば、アウトドアの疲れも癒されること間違いなし。 泉質はナトリウム硫酸塩泉。神経痛、筋肉痛、関節痛、うちみなどに効果があり、アウトドアで疲れた体を癒してくれる。近くを流れる狩野川は大変水がきれいなため、釣りをしたり川遊びをするには最適だろう。露天風呂は川のせせらぎが聞こえ、美しい緑とともに、日々の疲れを癒してくれる。
