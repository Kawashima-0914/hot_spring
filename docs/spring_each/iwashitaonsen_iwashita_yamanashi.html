岩下温泉（いわした）

所在地：山梨県

効能単純温泉

数多くの温泉がある山梨県で、1300年以上の歴史を持つ県内最古の温泉とされる温泉地、岩下温泉。現在は、明治時代初期に建てられた温泉旅館「岩下温泉旅館」で入ることができる。宿の施設は旧館と新館があり、旧館は、国の登録有形文化財。明治・大正期特有の様式である半地下構造の木造建築で、日帰りで入るほか宿泊してのんびり浸かることもできる。宿泊すれば、日本ワイン生産発祥地の山梨県で醸造した上質な国産ワインと、富士山麓牛や甲斐サーモンなどの地場産食材で作る会席料理も食べられる。「フルーツ王国」の山梨県らしく、周囲にはぶどう畑をはじめ果樹園が多く、季節ごとに旬を迎える果物狩りも楽しめる。 館内に源泉のある無色透明の単純温泉で、源泉の温度が28度と低めなのが特徴。加熱して利用する湯船と、源泉をそのまま使用している湯船の2種類があり、施設ではそれぞれを交互に入る方法をすすめている。神経痛、筋肉痛、冷え性、疲労などに効果があるとされる。
