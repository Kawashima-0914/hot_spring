竜王ラドン温泉（りゅうおうらどん）

所在地：山梨県

効能ラジウム泉

万病に効く「驚異の秘泉」と本で紹介されたこともある山梨県甲斐市の温泉地、竜王ラドン温泉。一軒宿の「竜王ラドン温泉 湯〜とぴあ」で入ることができる。効能豊かな湯を持つこの温泉は「医学の温泉」とも呼ばれ、湯治目的に訪れる人も多い。湯治のための長期滞在だけではなく日帰り入浴もできる。レトロな風情漂う建物は開設当時からのもの。今も活躍する芸能人が多く訪れた歴史もある。温泉療養・湯治向けの施設だけに、コインランドリーや各種アメニティ、備品の貸し出しなども行われている。名物の「ほうとうなべ」や「鶏もつ」など、館内では山梨県の郷土食も食べられる。 天然ラドンを豊富に含むナトリウム－塩化物・炭酸水素塩泉。糖尿病、じん麻疹、自律神経失調症、リウマチ、高血圧症、気管支炎、喘息、胃腸疾患、腹痛、痛風などの効能が期待できる。美肌効果もあるとされ女性にも人気だ。ぬる湯のため、ゆっくりと浸かることができ、体への負担が少ない。
