洞川温泉（どろがわ）

所在地：奈良県

効能単純温泉

奈良県大峯山・山上ヶ岳や女人大峯・稲村ヶ岳の登山口で、標高約820m余りの高地に位置する温泉地。大峯山は女人禁制の修験の山であり、門前町として夏期には山上ヶ岳の蔵王堂をめざす修験者や参詣者で非常に賑わう。反面、冬季は厳しい寒さに閉ざされる。なつかしい雰囲気のただよう街で、役行者の高弟「後鬼」の子孫の里とも伝えられている。また、一帯は代表的なカルスト地形（石灰岩台地）で、ごろごろ水に代表されるミネラル分を適度に含むおいしくて体に良い水も湧き、名水の山里はいつも賑わいを見せる。 弱アルカリ性単純温泉は、神経痛、リューマチ、筋肉痛、関節痛、運動麻痺、慢性消化器病、冷え症、疲労回復等に効能があり、肌にやさしい温泉のため、湯あたりしにくいのが特徴。
