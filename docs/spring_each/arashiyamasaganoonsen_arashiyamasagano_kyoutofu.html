嵐山嵯峨野温泉（あらしやまさがの）

所在地：京都府

効能ラジウム泉

京都市の中心部から北へ約15km、『鞍馬天狗』で有名な鞍馬の山里に湧き出す温泉地。明治中期までは、京から花背に抜ける鞍馬街道の宿場町として栄えた歴史あるエリアだ。現在では街並みが保全され、軒の低い田舎情緒あふれる景観が守られ続けている。また、鞍馬の自慢といえば京都屈指の高レベルを誇る水質の良さ。昔から「鞍馬の名水」と呼ばれるほど水のおいしさには定評があり、代表的な名水スポットに“牛若丸息つぎの水”と名づけられた霊水が湧出する「鞍馬寺」がある。 地下1,200ｍから湧出する源泉を利用した、ナトリウム・カルシウム塩化物泉。ナトリウムイオンとカルシウムイオンが豊富に含まれた湯は、皮膚の汚れをはがす作用があることから美肌効果が高い湯とされている。海水に近いナトリウム濃度であることから、様々な効能を持つ療養温泉としても使われている。
