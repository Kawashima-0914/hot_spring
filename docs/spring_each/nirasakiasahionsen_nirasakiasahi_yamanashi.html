韮崎旭温泉（にらさきあさひ）

所在地：山梨県

効能塩化物泉

地下の岩盤から湧き出る源泉を、加水・加熱・循環などの人工的処理をせず、かけ流しで堪能できる韮崎旭温泉。日本最大級の泡付きが楽しめる炭酸泉であることで知られており、効能が高い温泉として地元民に愛されている。この湯を楽しめるのは、温泉地と同名の日帰り入浴施設「韮崎旭温泉」のみ。豊富な湯量が、贅沢な気分にさせてくれる。館内には無料の休憩所あり。持ち込み可能なので、のんびりするならば軽食を持参しても良いだろう。この周辺は美しい自然や歴史ある見どころも多い。初夏には15万株ものレンゲツツジが咲き乱れる「甘利山」、夜の星を眺めるなら「銀河鉄道展望公園」、関東三観音の一つである「平和観音」、歴史のある「新府城跡」や白山城跡」など。温泉と一緒に楽しみたい場所が盛りだくさんだ。 ナトリウム-塩化物・炭酸水素塩温泉の泉質を持つ韮崎旭温泉。塩化系の塩分が汗の蒸発を防ぐ保温効果、炭酸水素塩泉による皮膚の洗浄作用・美肌効果が期待できる。湯は無色透明、入った瞬間から皮膚にたくさんの泡がつくのが大きな特徴。飲泉にも適しており、糖尿病や肝臓病にも効果があるとされ、ペットボトルに入れて持ち帰る人も多い。
