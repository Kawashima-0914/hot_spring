湯谷温泉（ゆや）

所在地：愛知県

効能塩化物泉

霊山鳳来寺山の麓に湧き出る源泉「鳳液泉」は鳳来寺開山の祖、利修仙人によって1300年ほど前に発見されたと伝えられており、その効能は万病に効くといわれている。 湯谷温泉は、渓谷「板敷川」沿いに9軒ほどの温泉旅館が立ち並ぶ静かな温泉郷で、露天風呂から眺める四季折々の風景は、都会の喧騒を忘れさせ癒しの時間へと誘う。 また、旅館自慢の会席料理は、鮎、山菜、しし肉、鳳来牛などの厳選された地元食材をふんだんに使用しており、この地ならではの旅をより鮮やかに彩る。 淡黄色の温泉は、成分濃度が高く、神経痛、関節痛、冷え症などに効果があります。
