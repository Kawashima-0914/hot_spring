武雄温泉（たけお）

所在地：佐賀県

効能炭酸水素塩泉

奇岩怪石で知られる蓬莱山（桜山）の山麓に佇み、「陶芸といで湯のふるさと」として年間約40万人が訪れる温泉地。竜宮を思わせる「楼門」は温泉街のシンボル。朱塗りの赤と漆喰の白が周囲の緑に映え、異国的な情緒を醸し出している。 市内には400年以上の伝統を持つ武雄焼の窯元が数多く点在。また、文禄・慶長の役の際、入湯に来た兵士に対して湯治客に迷惑をかけないようにと豊臣秀吉が示した入湯心得の古文書「豊臣秀吉塚崎温泉掟書」が保存されている。 歴史上名高い伊達政宗や宮本武蔵、伊能忠敬、シーボルトの入湯記録もある。 およそ1300年前、神功皇后が夢のお告げによって静養に訪れたときに発見したという霊験あらたかな開湯伝説が残る古湯。『肥前風土記』には「郡の西の方に温泉の出る巌（いわや）あり」と記されており、歴史の長さが伺える。泉質は、つるつる・すべすべの美肌効果をもたらす炭酸イオンが多く含まれ、無色透明のトロっとした肌ざわり。50℃前後で湧出するため、ほとんどそのままのかけ流しで利用されている。湯の花が多いのも特徴的。
