田牛温泉（とうじ）

所在地：静岡県

効能塩化物泉

「とうじ」と読む。南伊豆の中心・下田市の南部にある海沿いの地区。サンドスキーやシーカヤックなど、珍しいアクティビティが体験できることが特徴。波が岩を削って砂粒にし、サンドスキー場が自然に作られたというから驚きだ。シーカヤックができるのは「碁石が浜」という穴場のビーチ。急坂の下に広がる磯場は透き通るような色をしている。また、近くにある「竜宮窟」は伊豆の地形が作り出した神秘の洞窟だ。波の浸食によってハート型の穴が開いたもので、世界ジオパークを構成するジオサイトにも登録されている、近年注目の絶景スポットである。 田牛には宿が何十軒かあるが、温泉を引いている宿は一部であることに注意されたい。源泉温度は40度ほどとちょうどよく、海で冷えた体をじわじわと温めてくれそうだ。泉質は保温効果と保湿効果に優れた塩化物泉。海沿いの温泉らしい泉質である。温泉でも、太古の海の恵みを実感できるだろう。
