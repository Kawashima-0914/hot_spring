湯之谷温泉（鹿児島県）（ゆのたに）

所在地：鹿児島県

効能硫黄泉

霧島国立公園内に位置し、湯之谷川の上流に「天テラス（旧：霧島山上ホテル）」と「湯之谷山荘」の2軒存在する。山隔てた山間、湯之谷川に唯一湧く渓谷上流の源泉を、その昔は霧島山の修験者が利用し、その後、樵や農家が小屋掛けし湯を楽しんだといわれている。湯之谷山荘は1940年に湯治宿として開業した。木でできた鄙びた浴場も有名で秘湯マニアの中で知られる湯宿だ。近くには高さ23メートル、幅16メートルの豪壮華麗な滝「丸尾滝」や霧島有数のパワースポットとして知られる「霧島神宮」がある。大自然の神秘を感じに訪れてみては。 島津藩時代より、神経痛や切り傷、やけどなどによく効く湯治場として知られてきた「湯之谷山荘」は、自家源泉を5本所有し総湧出量は480L/分。自然の中に湧き出た温泉のため、気温・降水量・気圧などの変化により温泉の色彩・温度などが日々刻々と微妙に変化する。今日の湯の姿はもう二度と出会うことが出来ない。まさに一期一会だ。
