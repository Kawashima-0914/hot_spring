入広瀬ニュー浅草岳温泉（いりひろせにゅーあさくさだけ）

所在地：新潟県

効能ラジウム泉

浅草岳の山麓に広がる都市公園の只中で入れる宿泊施設の温泉「入広瀬ニュー浅草岳温泉」。全国にその名を知られた銘柄米・魚沼産コシヒカリの産地である南魚沼市の中でも大自然の緑に囲まれた大広瀬地区にあり、浅草岳、守門岳に囲まれた施設で、破間川の清流をのぞむ大自然を堪能できる。宿泊施設に宿泊した人のための食事は、清流が育んだ日本一の米と、魚沼市の旬の作物をふんだんに使う料理。春には花と緑と雪の里で「芝桜まつり」が開かれ、夏は月岡公園に10,000本のユリが咲き誇り、蛍が飛び交う四季の光景も美しい。 泉質はナトリウム-硫酸塩・塩化物泉（等張性弱アルカリ性温泉）。神経痛、筋肉痛、関節痛、肩こり、慢性消化器病、慢性婦人病、冷え性、きりきず、やけど、動脈硬化などへの効能が期待できる。鉄分豊かな湯が疲れを癒やしてくれると好評だ。
