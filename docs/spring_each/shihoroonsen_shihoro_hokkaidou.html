しほろ温泉（しほろ）

所在地：北海道

効能塩化物泉

手つかずの天然広葉樹林が広がる、自然豊かな地に佇むしほろ温泉。道の駅に併設されている施設「しほろ温泉プラザ緑風」が一軒宿だ。日帰り入浴も可能で、国際公認パークゴルフ場やレストランといったのレジャー施設、地場産品の販売コーナーなどがあり、地元民・リピーターに愛されている、しほろ温泉は茶褐色のモール温泉が大きな特徴で、美肌の湯としても知られ女性に人気。周囲の自然を思い切り堪能したいなら、士幌高原ヌプカの里での天体観測や、夏場は朝陽公園（ほたるの里）でホタルの観賞をするのがおすすめだ。 泉質は神経痛、筋肉痛、慢性皮膚病、慢性婦人病、切り傷などに効能があるとされるナトリウム塩化物泉。大きな特徴は北海道遺産としても指定されている植物性の「モール温泉（モール泉）」であること。琥珀色の湯は有機物を多く含み、肌への刺激が少ないのが特徴で、更に石鹸効果が強く肌がスベスベになる美肌の湯として知られている。天然の保湿成分も高く、女性にうれしい湯となっている。
