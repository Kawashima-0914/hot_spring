湯の鶴温泉（ゆのつる）

所在地：熊本県

効能硫黄泉

熊本県の南端に位置し、西は八代海（不知火海）に面し三方を山に囲まれた水俣市は2つの名湯を有する。 不知火海を望むリアス式海岸の美しい湯の児海岸と、熊本県と鹿児島県の県境に位置する緑深き山間に湧く、「新・日本百名湯」にも選出された湯の鶴温泉郷である。 平家の落人が、傷ついた鶴が湯浴みするのを見て湯の存在を知り、「湯の鶴温泉」と名付けられたとされる、開湯700年以上の歴史を有する伝統ある秘湯。この山間の湯の里では清流・湯出川も、時間もゆるりと流れる。 無色透明、大浴場の他岩風呂、展望露天風呂。
