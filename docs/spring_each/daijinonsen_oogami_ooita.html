大神温泉（おおがみ）

所在地：大分県

泉質炭酸水素塩泉、塩化物泉
効能神経痛、婦人病、冷え性、皮膚病、美肌

糸ケ浜海浜公園のあるリゾート地、糸ヶ浜の海岸沿いにあり、別府湾を一望できるリゾートホテルの自家源泉で作られた温泉。町内は大分県内でも指折りの「名水の郷」であり、海沿いでありながら豊かに湧く温泉もこの「名水」の恩恵を受けている。別府湾を臨む絶景の温泉であり、観光客のみならず地元の人にも親しまれている。海水浴場である糸ケ浜海浜公園の敷地内には、野球場やテニスコート、そしてキャンプ場などがあり夏には賑わう。また、温泉地からほど近い場所にはハーブ園とローズガーデンのある「おおがファーム」もある。別府湾の水で遊び、ハーブやバラに癒やされた後、温泉に入れば、極上の癒しを手に入れられるはずだ。 塩化物泉と炭酸水素塩泉の2種類のお湯がある。塩化物泉は口に含むと感じる塩気が特徴。入浴では痛みを和らげる効果などが期待できるため、切り傷や皮膚病、神経痛や婦人病などに効果的とされている。また、炭酸水素塩泉は肌を綺麗にするクレンジング作用があるとされるお湯。入れば肌がスベスベになる美人の湯とも呼ばれている。
