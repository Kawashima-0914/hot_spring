大村温泉（おおむら）

所在地：鹿児島県

効能硫酸塩泉

里山が広がる田園の村、祁答院町（けどういんちょう）の旧町役場があった場所に近い温泉地。周囲を緑に囲まれた静かな温泉地には町営の温浴施設があり、周囲に住む人々の憩いの場としても親しまれている。また、祁答院町は「ほおずき」の産地。夏には特産品販売所で「ほおずき市」が開催され、鮮やかな朱色が町内を飾る。販売所では、地元の農産物を使用した加工品や、新鮮な野菜などを購入することもできるので、祁答院町の夏の風物詩になっている「ほおずき市」の時期に訪れることができたら、ほおずきと一緒に地元の味を手にとってみると良いだろう。 泉質は透き通った湯が特徴の硫酸塩泉。浴用としては神経痛、筋肉痛、関節痛、五十肩など、飲用としては胃腸病や便秘に効能があるとされている。入浴施設の中には、湯を販売しているところもある。湯を毎日購入しポリタンクで持ち帰って、自宅の湯船で温泉気分を堪能する固定ファンもいるそう。
