西合志温泉（にしごうし）

所在地：熊本県

効能塩化物泉

海岸付近に湧くことの多い、ミネラル豊富な温泉が湧き出る温泉地。「きしゃぽっぽ」という可愛らしい名前の温泉施設が提供している。10棟の部屋に備え付けられた半露天風呂は全て貸切専用。部屋ごとに作りが違うので、泊まる部屋が違えばお風呂の様式も違う。貸切風呂なので家族連れやカップルなども利用しやすい。合志市野々島には、全国の石器製作遺跡の中で唯一の国指定史跡となっている「二子山石器製作遺跡」や、いちご狩りやぶどう狩りができる高司農園などがあり、観光地としても楽しめる。観光地の中の貸切温泉という贅沢な環境を楽しみたい。 貸切専用のお風呂はすべて、天然温泉100％掛け流し。ラドン含有でミネラル豊富な含放射能-ナトリウム-塩化物泉が特徴。貸切なので湯の量と温度は調節できる。やけどや切り傷、皮膚病や婦人病、痛風や高血圧症など、様々な疾患に効能があるとされている。貸切風呂の窓からはそれぞれに異なる自然の風景が見え、四季折々の風情も楽しめる。
