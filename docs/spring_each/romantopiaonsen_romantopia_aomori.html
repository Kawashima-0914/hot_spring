ロマントピア温泉（ろまんとぴあ）

所在地：青森県

泉質二酸化炭素泉、炭酸水素塩泉、塩化物泉、硫酸塩泉
効能神経痛、筋肉痛、関節痛、冷え性、高血圧、痔疾

世界遺産「白神山地」に近く、雄大な自然に囲まれたロマントピア温泉は、弘前から約15分ほどの場所にあり、周辺へのアクセスも良く、観光スポットへの拠点としても最適。 バーベキューハウスやテニスコート、森林科学館なども併設する「星と森のロマントピア」の敷地内には、100％源泉のかけ流しの温泉が湧く。豊富な湯量を誇り、泉質は、肌に優しいナトリウム泉とカリウム塩化物泉で、神経痛や筋肉痛、疲労回復に効果的とされ る。 露天風呂からは、津軽富士と呼ばれる岩木山を一望でき、夜は満天の星を眺めることができるという魅力がある。街の喧騒から離れ、鳥のさえずりに耳をかたむけながら朝の高原を散策したり、アウトドアでわいわいと思い出を作ったり、目的に応じてオールシーズン満喫できる多彩な森のステージである。 天井が高く開放的な内風呂からも「岩木山」を望める。
