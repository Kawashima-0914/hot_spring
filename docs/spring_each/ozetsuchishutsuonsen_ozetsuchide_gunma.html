尾瀬・土出温泉（おぜ・つちで）

所在地：群馬県

効能単純温泉

ミズバショウの花咲く光景をうたう歌曲『夏の思い出』の舞台は、群馬県片品村の高層湿原、尾瀬ヶ原。片品村にはウインタースポーツがメインのリゾート地として人気の尾瀬岩鞍高原もある。尾瀬・土出温泉は、この高原のスキー場に隣接するリゾートホテルの大浴場で入れる温泉。源泉である尾瀬岩鞍高原の湧出泉「市山の湯」から引いており、この源泉の名で呼ばれることもある。レトロ感覚にもあふれるホテルのほかに、この源泉を引いている施設は村内に点在しており、宿泊での入浴も日帰り入浴も可能。スキーで疲れた体の癒やしも美肌効果も期待できる。 土出温泉の泉質はアルカリ性単純温泉で、神経痛や疲労回復、冷え性まで幅広い効能が期待できる。アルカリ性のため美肌効果も高いとされる。女性にうれしい泉質も持つ。宿泊施設の中には日帰り入浴ができるところもあり、旅の途中で源泉市山の湯を楽しむこともできる。
