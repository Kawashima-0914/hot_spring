湯元新祖谷温泉（ゆもといんいや）

所在地：徳島県

効能硫黄泉

徳島県三好市にある大歩危祖谷温泉郷のひとつで、かずら橋にも近い場所に一軒宿の「ホテルかずら橋」が存在する。過疎対策として、地元の有志により1988年に開湯した温泉地だ。「かずら橋」は周囲を木々に囲まれ、眼下には祖谷川の渓流を見下ろす日本三奇橋の一つ。約800年前に源平の戦いに敗れた平家一族が、剣山、平家の馬場での訓練に通うため架けられたと言われており、長さ４５メートル、幅２メートル、川面からの高さは１４メートルにもなる。一歩踏み出すたびにきしんでユラユラと揺れる橋はスリル満点！春は藤が咲き、夏は新緑、秋は紅葉に染まり、冬は見事な雪景色を魅せる「かずら橋」で四季折々の景観を楽しんでみては。 泉質は単純硫黄泉。関節痛・筋肉痛・五十肩・運動麻痺・関節のこわばり・慢性消火器病・痔疾・冷え性・疲労回復・健康増進など効果が期待できる。「ホテルかずら橋」には大浴場、露天風呂のほかに、敷地内のケーブルカーでのぼって行く天空露天風呂がある。天空露天風呂からは、祖谷の深い谷と山々を見渡す絶景が広がり、その美しい景色に思わず息を飲んでしまう。
