城山温泉（しろやま）

所在地：鹿児島県

効能炭酸水素塩泉

国分平野が一望できる霧島市国分の高台にある温泉地。地下1200メートルから湧出する源泉をかけ流しで楽しむことができる、国分エリア随一の温泉でもある。温泉地から最寄駅の国分駅にかけてのエリアは、商業地が多い地域。ビジネスや観光などの宿泊地としても便利である。温泉と並んでこの街で提供されるのが、霧島市上小川の山麓に湧出する「筒之口名水」。江戸時代から酒の製造に使用されるほどおいしい水として知られている。温泉への入浴客だけではなく近隣住民も生活水として利用する。温泉地近くにはソメイヨシノと八重桜の美しい観光地、城山公園がある。春には温泉と共に花見に訪れるのもおすすめ。 塩化物泉の泉質を持つ城山温泉。湯量も豊富に湧出する黄金色の湯は、塩分と鉄分を豊富に含んでおり、塩味が感じられる。入浴すれば、きりきず、やけど、皮膚病など、飲用すれば慢性消化器病、糖尿病、通風、肝臓病などに効能があるとされている。
