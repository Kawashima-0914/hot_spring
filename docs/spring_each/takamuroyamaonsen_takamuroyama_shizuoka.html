高室山温泉（たかむろやま）

所在地：静岡県

効能硫酸塩泉

東伊豆の富戸(ふと)にある温泉地。大室山のふもとに広がる温泉地で、麓から山頂まではリフトが通っている。大室山は今から約4000年前の噴火によって作られた山であるため、山全体が四季の移ろいとともに様々な表情を見せてくれるのが美しい。近くにはゴルフ場や伊豆シャボテン動物公園もあるためレジャーにももってこい。伊豆シャボテン動物公園には、名前の通り世界中のサボテンだけでなく、冬の露天風呂に入浴するカピパラをはじめ、サル、カンガルー、鳥類を展示する動物公園がある。カピバラと一緒に入浴することはできないが、風呂に浸かるカピバラを見たあとに温泉に浸かるのもなかなかできない経験だろう。 泉質はカルシウム―硫酸塩泉。肌に潤いを与え、保温効果もある。キズに良いといわれる泉質で、無色透明のまろやかなお湯だ。レジャーで1日遊んだ後の体をゆっくりと癒して、あたためてくれるだろう。カピバラと一緒に入浴することはできないが、気づけばカピバラのようなまったりした表情になっているかもしれない。
