浮島温泉（ふとう）

所在地：静岡県

効能硫酸塩泉

西伊豆の温泉地で、海がほど近く夕日が美しい。「恋人岬」の名で有名な岬があり、富士山や駿河湾を一望にする絶景の岬に、愛の鐘「ラブコールベル」が設置されている。「この鐘を3回鳴らしながら愛しい人の名を呼ぶと愛が実る」と言われており、まさに恋人の聖地ともいえるだろう。一帯は「伊豆半島ジオパーク」に登録されている自然豊かな地だ。近くの岩場では日本でも大変珍しい、干潮時のみ岩場が渡れる「トンボロ現象」を見ることができ、さらにタイミングが良ければ「トンボロ渡り」を体験できる。砂遊びやカヤック、磯遊び、ダイビングなどもできる。 泉質は肌に潤いを与えてくれる硫酸塩泉。ナトリウムやカルシウムなど、保温効果のあるミネラルも豊富だ。アルカリ性の湯は肌を包みこんでスベスベにしてくれる。仁科の源泉井戸からこんこんと湧き出る浮島温泉を贅沢にかけ流しされた湯船に浸かれば、アクティビティを満喫して疲れた体を癒してくれるだろう。
