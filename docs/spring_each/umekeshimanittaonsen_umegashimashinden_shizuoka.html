梅ヶ島新田温泉（うめがしましんでん）

所在地：静岡県

効能炭酸水素塩泉

静岡市葵区にある温泉地。静岡市内にありながら標高800mの高原に位置し、豊かな自然を楽しむことができる。やまめ釣り・つかみ取り体験ができる「魚魚の里」が近くにあり、体験を楽しんだ後は新鮮な魚をいただくことができる。静岡と山梨の県境をなす安部峠（標高1416m）も近くにある。戦国時代、武田信玄が金を甲府まで運ばせたという。四季折々の景観が楽しむことができ、初夏にはツツジが美しく、秋には紅葉狩りを楽しむことができる。静岡側は舗装がされ、例年5月～12月初旬まで、林道を自動車でも通行できるため、ドライブに出かけるのもよいだろう。 泉質はナトリウムー炭酸水素塩温泉。お湯は無色透明でトロッとした感触が特徴的である。まろやかでとろみのあるお湯のおかげで、湯上がりは肌がしっとりすべすべになる効果も期待できる。主成分の重曹は、肌の角質を落としてなめらかな肌に導いてくれることだろう。切り傷やヤケドなどの適応症もある。
