座禅温泉（ざぜん）

所在地：群馬県

効能胃腸病、冷え性、動脈硬化、皮膚病、打撲、痔疾、美肌

年間200日滑走可能なスキー場「丸沼高原」のゲレンデに湧く温泉で。冷えて疲れた体をよく温めて癒やす温泉として、スキー客はもとより、周辺の百名山の登山客にも親しまれている。丸沼高原には、オートキャンプ場やアスレチックが楽しめるツリーアドベンチャー、夏にはバーベキューハウスもオープンする。また、高原では散策や登山のための、さまざまなコースを用意。「燧ヶ岳」を眺望できるコースをはじめ、六地蔵や大日如来などをめぐるコースなど、初心者でも気軽に山歩きを楽しめる散策路があるのも大きな魅力だ。ロープウェイもあり、山頂のカフェではパスタやフレンチトーストなどといったおしゃれなメニューも提供している。 泉質は無色透明で苦味のある、日本では数少ないナトリウム・カルシウム-硫酸塩・炭酸水素塩温泉。きりきず、やけど、慢性皮膚病、動脈硬化症などに効能が高いとされる。また、泉質がよく美肌効果も期待できる温泉として親しまれている。
