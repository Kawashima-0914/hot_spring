寒水沢温泉（かんすいざわ）

所在地：青森県

効能塩化物泉

八甲田の西、「青森駅」よりバスで約60分、八甲田ロープウェー山麓駅まで徒歩約1分という場所に位置する、「八甲田山リゾートホテル」は、一軒宿の温泉地。 八甲田山スキー場のゲレンデの真下というロケーションから、冬は多くのスキーヤーで賑わいを見せる。標高約1326mの田茂萢岳の山頂まで走る八甲田ロープウェーも近く、八甲田山の景色はもちろん、北海道の景色まで望める眺望は圧巻だ。 秋には八甲田山が赤いじゅうたんで一面覆われたように紅葉で真っ赤に染まり、観光客を魅了するのだとか。四季折々の美しい自然、晴れ上がった夜空に瞬く満天の星、八甲田山の眺望を眺めながら湯浴みを楽しむのも、また格別だ。素晴らしい環境の中で、まろやかな泉質の温泉を満喫。 八甲田山スキー場のゲレンデ下に湧く温泉。開湯は平成元年と新しく、八甲田山地区では新興の温泉だ。心持ち白く濁っているように見えるが、透明度が高い源泉は肌触りが良く、湯上りは肌がすべすべとして心地よいと評判。泉質はナトリウム‐炭酸水素塩・塩化物泉で、リウマチ、やけど、慢性皮膚病などに効能を発揮する。外来入浴できるので、八甲田山を散策がてらぶらっと入浴していく人も多い。
