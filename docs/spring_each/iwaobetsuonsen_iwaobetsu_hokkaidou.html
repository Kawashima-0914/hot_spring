岩尾別温泉（いわおべつ）

所在地：北海道

泉質炭酸水素塩泉、塩化物泉
効能神経痛、リウマチ、疲労、ねんざ、冷え性、皮膚病、痔疾

岩尾別温泉は知床半島・羅臼岳の北麓に位置する原生林に囲まれた温泉地。岩尾別というのはアイヌ語で「硫黄の川」という意味で、古くからアイヌの人に知られる温泉地でもあった。羅臼岳は登山客が多く訪れる山でもあり、日本百名山の一つにも数えられている。険しい登山道で野生のクマやエゾ鹿と遭遇する可能性もあるので、しっかりと対策をして臨みたいところだ。ホテルのレストランや周囲の食事処では、オホーツク海で育ったカニやホタテなどをはじめとした海の幸を堪能できる。訪れた際は、旬の海鮮と温泉で日頃の疲れを癒やしたい。4月下旬頃から11月上旬ごろまでの営業であるため、事前に要確認だ。 ナトリウム・カルシウム-塩化物・炭酸水素塩泉の泉質を持つ温泉で、日によってターコイズブルーやグリーンなどの幻想的な色を見せる。リウマチ、神経痛、筋肉痛、関節痛、五十肩、うちみ、くじき、慢性消化器病、冷え性、きりきず、皮膚乾燥症など多くの効能を持つとされる。とろみのある青みがかかった湯が特徴で、体を包みこむような湯ざわりを堪能できる。
