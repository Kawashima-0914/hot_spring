天恵泉（てんけいせん）

所在地：山梨県

合併前は「白根町」であったことから、「白根に湧く天の恵みの湯」と呼ばれる、雄大な自然に囲まれた温泉地、天恵泉。日帰り入浴施設の「天恵泉白根桃源天笑閣」では、貴重な「木曽さわら材」を使った浴槽で入ることができる。日本でも有数の水素イオン濃度を誇る高アルカリ温泉を、湧出温度そのままの源泉槽と、温度を上げた加温槽で交互に入る「温冷交互浴」により、さらに高い効能を得られるとされる。豊かな自然の中にある温泉は、秋に訪れれば美しい紅葉を眺めながら温泉を楽しめる。温泉スタンドも設置されており、温泉を購入して自宅で楽しむこともできる。 アルカリ性単純温泉（低張性アルカリ性低温泉）の泉質で、神経痛、筋肉痛、冷え性、疲労などの広い効能が期待できる。国内有数の水素イオン濃度であるpH10.6の高アルカリ性温泉は、肌に優しく美肌効果も高いとして女性にも人気。
