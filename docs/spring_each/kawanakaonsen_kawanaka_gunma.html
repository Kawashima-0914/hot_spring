川中温泉（かわなか）

所在地：群馬県

効能硫酸塩泉

川中温泉は雁ヶ沢川沿いに位置する閑静な温泉地で、和歌山県の龍神温泉、島根県の湯の川温泉と並び「日本三美人の湯」のひとつ。美肌効果が期待できるこの湯は、一軒宿「かど半旅館」のみで入浴できる。「川中」の名の通り、旅館の目の前の川底から厳選が自然湧出しており、鎌倉時代以前に発見されたとも言われている。源泉温度は約35℃とぬるめで、1回の入浴を30分～1時間として、出たり入ったりを繰り返すことで、より効能を感じられる。日帰り入浴はできないため、休暇を取って訪れよう。夕食では、名物の「おきりっこみ」を堪能できる。 カルシウムを含む硫酸塩泉の泉質で、皮脂を洗い流し、美肌への効能が高いとされる美人の湯。とくに肌を白くする効果があるとされ、女性に高い人気を誇る。そのほか神経痛や筋肉痛、冷え性や疲労などの広い効果も期待できる。湯あたりしにくい湯だが、体調に注意しながら入浴したい。
