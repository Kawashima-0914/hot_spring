屋久島温泉（やくしま）

所在地：鹿児島県

効能単純温泉

面積約504.29k㎡、日本で7番目に大きな島となる鹿児島屋久島。屋久杉が自生する森林を含めた島の面積の約21％がユネスコの世界遺産に登録されている。 ネイチャーウォークやマリンスポーツを目的とした根強いファンが多く、全国から多くの観光客を集めている。屋久島温泉は屋久島空港より徒歩約1分という空路組には嬉しい立地にある温泉地だ。 リラクゼーション施設が充実しているほか、「観光ベースキャンプ」と称して、トレッキング、ダイビング、フィッシング、自然観察（ウミガメの産卵など）など、各種アクティビティの相談にも乗ってくれるサービスもある。森と思い切り遊んだ後は温泉で身体を癒してあげるのも忘れずに。 pH9.44、循環ろ過式・加温。日帰り入浴可能。
