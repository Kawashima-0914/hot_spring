立久恵峡温泉（たちくえきょう）

所在地：島根県

効能塩化物泉

島根県出雲市の南部にある県立自然公園「立久恵峡」のすぐそばに湧き出る温泉地。宿泊施設、日帰り入浴施設があり、立久恵峡を間近に眺めながらの湯浴みができるダイナミックな露天風呂が評判を集めている。「立久恵峡」は、神戸川の清流にそって約2kmにわたり高さ100～200mもの岩壁がそそり立つ。この集塊岩の渓谷は神戸川の浸食や風化によってできたもので、左岸には神亀岩、烏帽子岩、ろうそく岩などの名で呼ばれる見事な奇岩が連なっている。下流にかかる不老橋から浮嵐橋の間は自然観察モデルコースとして遊歩道が設置されており、途中には展望台や五百羅漢、霊光寺などの見所もたくさんある。橋から橋まで回遊すると1周約1時間。鳥のさえずりを聞きながらの自然散策もお楽しみ。 ここ立久恵にはリュウマチや神経痛などに効く良質の温泉があり、その昔、この辺りが海の底だった名残りで塩分を多く含み、さらっと浸透する肌触りが特徴。「山陰の耶馬渓」と呼ばれ国の名勝天然記念物に指定されている一大渓谷「立久恵渓」は四季折々に美しい表情を見せ、耳をすませばかじかの鳴き声も聞こえる。そんな自然豊かで贅沢なひとときが過ごしてみては。
