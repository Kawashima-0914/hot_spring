北広島温泉（きたひろしま）

所在地：北海道

効能塩化物泉

札幌市街や新千歳空港から車で約40分、リゾート気分をたっぷり味わえる地に湧くのが「北広島温泉」。付近にはゴルフ場が多く点在し、札幌ドームやスキー場もあり市街から来るレジャーを楽しむ人々の拠点としても利用 されている。 札幌農学校教頭のクラーク博士が明治10年に帰国の際に立ち寄って、かの有名な「青年よ大志を抱け」という言葉を残したのはここ北広島である。「自然の森キャンプ場」や「えこりん村」など大自然や動物 とのふれあいを満喫できる施設もあり、ファミリーが余暇を楽しむ風景が多く見られる。 「千歳アウトレットモール・レラ」や「三井アウトレットパーク札幌北広島」など商業施設が充実しているのも嬉しい。レジャーを思い切り楽しんだ後、その疲れを良質な温泉でゆっくり癒したい。 札幌市街より車で40分ほどの地に湧く「北広島温泉」は、ゴルフ場やスキー場が多く点在するリゾート地に建つ「北広島クラッセホテル」で入浴することができる。ナトリウムー塩化物泉で、神経痛、筋肉痛、関節痛など効能も多岐に亘るお湯は、黄褐色の色が特徴的でとろりとした肌触りが心地よく、美肌効果も高いと女性に人気が高い。森林を目の前にした露天風呂での湯浴みはマイナスイオン効果も手伝い、日頃の疲れを一層癒してくれるに違いない。
