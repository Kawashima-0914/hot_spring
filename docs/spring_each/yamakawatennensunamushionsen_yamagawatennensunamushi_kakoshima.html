山川天然砂むし温泉（やまがわてんねんすなむし）

所在地：鹿児島県

効能塩化物泉

山川天然砂むし温泉は、その名のとおり湯ではなく「砂湯里」という天然の砂に入る砂風呂。開聞岳や大隅半島などを望める伏目海岸は、温泉がいたるところで自然に沸き立つ場所であり、砂浜に掘った穴へ体を埋めてもらうことで、温泉熱を全身に浴びる入浴を楽しむことができる。サウナに入ったように、10分～15分で砂の温度と圧力により全身から汗が噴き出し、出た後は体が軽くなる感覚があるほど。伏見海岸一体はその美しさから景勝地としても、また温泉リゾートとしても楽しめる。 砂風呂には温泉の成分が含まれており、ナトリウム-塩化物泉の性質を持っている。神経痛や腰痛、アトピーや胃腸病、また便秘や肥満など多くの効能が期待できる。また、天然保湿成分のメタケイ酸が全国屈指の含有量を誇り美肌効果が高いことで女性から人気がある。湯を楽しむ温泉とはまた違う感覚を、砂むし温泉で体験してみてはいかがだろうか。
