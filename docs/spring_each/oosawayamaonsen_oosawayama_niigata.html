大沢山温泉（おおさわやま）

所在地：新潟県

効能炭酸水素塩泉

日本一の評価を受ける銘柄米「魚沼産コシヒカリ」の生産地、新潟県南魚沼市。雪深い街は人気の「大沢山温泉」が湧く地でもある。大沢駅から徒歩で約6分、上越国際スキー場の大沢ゲレンデにも近い温泉地で、周囲を自然に囲まれているので、夏になると緑豊かで閑静な温泉地になることから人気を呼んでいる。江戸時代初期の寛永期に開業したと伝えられる歴史の長い温泉地は「秘湯」とされ、3件の宿が営業している。風情豊かな温泉宿は、シーズンには予約が取りにくくなるほど人気が高い。 泉質はナトリウム-塩化物炭酸水素泉(アルカリ性低張性温泉)。神経痛、関節痛、冷え性などのほか、きりきず、やけど、慢性皮膚病、美肌など肌への効能が高いとされている。無色透明の湯は、つるつるとした感触で肌によいと女性に人気。
