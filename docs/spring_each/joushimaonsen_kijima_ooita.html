城島温泉（きじま）

所在地：大分県

効能単純温泉

阿蘇くじゅう国立公園内に広がる城島高原に位置する温泉地。別府温泉まで車で約20分、由布院温泉まで約15分とちょうど中間に位置しており、「奥別府」として親しまれている。間近に望む雄大な鶴見・由布岳、静かに佇む志高湖。 大自然に抱かれた高原にはゆったりとした時間が流れ、疲れた心と体をリフレッシュするのにおすすめだ。この高原リゾートの中心にあるのが、一軒宿の「城島高原ホテル」。 豊富に湧き出る温泉を利用した入浴施設はもちろん、遊園地やゴルフ場などの総合レジャー施設とも隣接しており、様々な楽しみ方ができるのも魅力。 標高約700ｍの城島高原に建つ「城島高原ホテル」に豊かに湧出する温泉。地底約1500mの源泉から約62℃前後の温泉が毎分約230L湧き出ている。ほのかな湯の香りと滑らかな肌触りを楽しめ、美肌の湯としても人気がある。桧造りの男性風呂と大理石造りの女性風呂の内湯と男女それぞれ露天風呂も備えている。
