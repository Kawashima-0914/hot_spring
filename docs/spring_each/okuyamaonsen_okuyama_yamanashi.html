奥山温泉（おくやま）

所在地：山梨県

効能単純温泉

静かな山あいに位置し、美しい大自然に囲まれた温泉地。付近のオートキャンプ場からや、登山帰りの客が、心身の疲れを癒やしに訪れる。温泉と同名の温泉施設は、週末・祝日のみ宿泊できる。ゆっくり過ごしたいなら週末に訪れるのがおすすめだ。露天風呂からは四季折々の山景色を望み、秋の紅葉は格別。また、自然に囲まれた地域ならではの観光スポットも数多く存在する。福士川渓谷にある七つ釜は、幾重にも重なった滝つぼが複雑で壮大な景観を見せる。また、春に訪れるのなら本郷の千年桜は外せない。南部町本郷の旧妙善寺境内に咲く千本桜で、樹齢500～600年という老樹が花を咲かせた姿は幻想的だ。 肌をなめらかにしてくれるアルカリ性単純温泉（低張性アルカリ性高温泉）。無色透明で匂いが無いため、入りやすい泉質である。2つある露天風呂は、1週間ごとに男女入れ替えが行われ、片方がぬるめに設定されている。神経痛、筋肉痛、運動麻痺、慢性消化器病、痔疾、冷え性、疲労回復など、効能がさまざまにあるのも特徴だ。
