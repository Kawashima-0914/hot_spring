柳川温泉（やながわ）

所在地：山形県

泉質塩化物泉、硫黄泉
効能神経痛、婦人病、冷え性、動脈硬化、皮膚病、糖尿病、便秘

朝日連峰の麓に位置し、「熱の湯」と呼ばれるほど保温効果が高いとされる湯が湧く地。入浴施設は「奥おおえ柳川温泉」のみ。平成7年にオープンしたが、東日本大震災の影響で温泉の自噴が止まったため、平成23年に新たに掘削されている。周囲の観光としておすすめなのは、多くの画家や写真家が訪れるというビューポイント「楯山公園」や、紅葉が色鮮やかな「大頭森山自然公園」などがある。また、県指定天然記念物である「神代カヤ」や「松保の大杉」なども見ものだ。大自然に抱かれた温泉地でゆっくりと過ごしたい。 温泉の泉質はナトリウム-硫酸塩・塩化物温泉で主に動脈硬化症、きりきず、やけど、慢性皮膚病などに効果があるとされている。また飲泉も可能で、飲用としては慢性胆嚢炎、便秘、肥満、虫歯予防への効能が期待できる。保温効果が高いとされる泉質で、「熱の湯」と呼ばれるほど、体がよく温まり冷めにくい。
