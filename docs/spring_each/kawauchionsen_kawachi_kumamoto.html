河内温泉（かわち）

所在地：熊本県

効能単純温泉

有明海に突き出した小さな岬に位置する温泉地。肥後藩主細川氏が新地開拓の際に発見したと伝えられている。河内温泉のある河内町には、「ナルシストの丘」と呼ばれる絶景ポイントが存在する。有明海と雲仙普賢岳を一望できることから、ドライブやサイクリングなどで立ち寄る場所として最適。海と山にも囲まれた温泉地なので、宿泊施設などでも新鮮な海の幸や山の幸を楽しめる。また、河内町はみかんの産地。温泉の周辺には多くのみかん農園がある。美しい景色を見て温泉に入ったら、海と山の幸、そしてご当地のみかんを食べたい。 河内温泉の泉質は自律神経不安定症、不眠症、うつ状態などに効能があるとされる弱アルカリ性、低張性の単純温泉。温度の低い低温泉なので加熱のみ行い、加水はしないので、ほぼ源泉100％のお湯を楽しめる。有明海を目前にした温泉地で、自然の光景を見ながらゆったりと浸かれる。
