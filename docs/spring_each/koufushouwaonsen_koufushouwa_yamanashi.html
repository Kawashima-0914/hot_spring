甲府昭和温泉（こうふしょうわ）

所在地：山梨県

効能塩化物泉

温泉地の多い山梨県の中では珍しい、ビジネスホテルの中の天然温泉「甲府昭和温泉」。ビジネス用途の宿泊者も、日本一の渓谷美を誇る「昇仙峡」やワインセラー見学など甲府市内の観光名所を訪れた人も、自噴の源泉かけ流し天然温泉に入れることから、1日に100人以上が訪れる人気温浴施設でもある。ホテル宿泊者でなく日帰りでも入浴可能。入ると体が気泡に包まれ、体の芯まで温まる。また、山梨はぶどうをはじめとした果物やほうとうなどの郷土料理など、グルメも楽しめる地域。ホテルの中には地元・山梨県の名物などを食べられる食堂もあるほか、周辺にはフルーツ狩りを楽しめる場所も点在している。 わずかにぬめりのある茶褐色のナトリウム一炭酸水素塩・塩化物温泉。湯量豊富な低張性の弱アルカリ性で、神経痛、関節痛、肩こり、慢性消化器病、痔疾、冷え性、疲労回復、きりきず、慢性皮膚病などへの効能が期待できる。
