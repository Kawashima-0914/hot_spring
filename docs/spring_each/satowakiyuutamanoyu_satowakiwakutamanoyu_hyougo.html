さとわき湧玉の湯（さとわきわくたまのゆ）

所在地：兵庫県

効能単純温泉

兵庫県の中央部からやや南東に位置する加東市。瀬戸内海性気候のため、一年を通して過ごしやすい温暖な気候である。 西にはJR加古川線と加古川が、東には住宅地と工業団地を一体化したニュータウンが広がり、高野山真言宗に縁が深い寺院や札所が点在している。さとわき湧玉の湯は、豊かな緑に包まれた丘陵地に位置し、隣接する三木市の里脇ファームランドから毎日タンクローリーで運び湯をしている温泉だ。 新鮮な温泉を楽しめるところが嬉しい。周囲には東条湖・ゴルフ場・キャンプ場・おもちゃ王国、夏には見事なひまわりが咲き誇るひまわり公園などの施設が満載。思いきり身体を動かしたら、多彩なお風呂を完備した温泉でのびのびと寛ごう。 寝湯やジャグジーをはじめとした5つのお風呂の湯巡りが楽しい。温泉を使用しているのはそのうち2つ。日帰り入浴可能。
