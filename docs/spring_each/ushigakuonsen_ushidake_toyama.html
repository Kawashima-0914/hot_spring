牛岳温泉（うしだけ）

所在地：富山県

効能塩化物泉

富山県の県庁所在地で、富山観光の拠点となる富山市。景勝地、城趾、神社仏閣、レジャー、祭事、特産品など非常に多くの観光資源に恵まれている。温泉地は市内だけでも9カ所以上あり、牛岳温泉もその一つ。 牛岳のふもと、日本海を内陸に入った標高約300mの山岳地に位置し、大自然の中にあり、富山平野を一望できる温泉。周辺でスキーやキャンプを楽しむ観光客に愛されている。 日帰り入浴なら牛岳温泉健康センターで入れるが、敷地内にある宿泊施設でもここのお湯が楽しめる。さまざまな効能を持つ温泉をゆっくり堪能したい。キッチン付きのコテージは、家族旅行や団体旅行に最適だ。砺波や八尾にも近いのでマイカーなら行動範囲も広がるだろう。 ナトリウムーカルシウム塩化物泉、源泉温度60.0℃。全身浴、露天風呂、寝湯、打たせ湯、サウナあり。大浴場は朝6時にお湯を張り替える。
