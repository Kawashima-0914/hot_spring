小金湯温泉（こがねゆ）

所在地：北海道

効能硫黄泉

人気の温泉地、定山渓から少し離れた地にある小さな温泉地、小金湯温泉。豊平川が流れる山の麓に、明治時代に開かれたとされている。温泉宿は、「湯元小金湯」と「湯元 小金湯温泉まつの湯」の2軒。開湯後は効能が高いと話題になり、その効能を求めて多くの人が押し寄せたと言われる。樹齢700年の「桂の木」はこの静かな温泉地のシンボル。根本に不動尊が祀られており、この不動尊は「桂不動」と呼ばれる。秋には紅葉が楽しめる風光明媚な温泉地としても名高い。周辺は秋になると紅葉が美しく、アイヌ民族の心と文化を伝える施設「札幌市アイヌ文化交流センターサッポロピリカコタン」は徒歩圏内。保養の地で、良湯も観光も楽しめる。 無色透明で硫黄の香りが漂う単純硫黄泉で、皮膚病、神経痛、運動麻痺、うちみ、くじき、慢性消化器病、痔疾、冷え性、慢性婦人病、きりきず、糖尿病などさまざまな効能が期待できる。毛細血管を広げ、血圧を下げるとも言われており、古くから湯治場として親しまれている。
