高山温泉（たかやま）

所在地：群馬県

効能塩化物泉

「星を見るために訪れる人が日本で最も多い」と言われる、群馬県高山村の天文台「県立ぐんま天文台」。美しい星空に近づける村には、豊かな自然の中に高山温泉が湧く。入浴施設としては「高山温泉ふれあいプラザ」と「高山温泉いぶきの湯」があり、どちらも日帰り入浴ができる。露天風呂では「県立ぐんま天文台」で眺めることのできる星空を見ながらお湯に浸かれる。天文台でも夜間に天体観測などのイベントが開催され、その美しい星空を体験することができる。ほかにもスコットランドの古城を移築し復元した「ロックハート城」では、中世の町並みを再現した協会やレストランのほか、ドレスを着て撮影をするプリンセス体験も人気。 泉質は体が芯から温まり、冷めにくいという特徴を持つナトリウム・カルシウム・塩化物温泉（弱アルカリ性低張性高温泉）。きりきずややけどなどの治癒に効果があるとされるほか、慢性皮膚炎や美肌にも効能があるとされている。入浴後は肌がスベスベになることから、女性にも人気がある。
