寺泊温泉（てらどまり）

所在地：新潟県

泉質炭酸水素塩泉、塩化物泉
効能にごり、にごり（褐色）、神経痛、筋肉痛、関節痛、疲労、冷え性、高血圧、糖尿病、痔疾、美肌

県下第2位の人口をようし、長岡都市圏の中核市となる長岡市。江戸時代には長岡藩の城下町として栄えた。城跡や陣屋跡などの名所旧跡をはじめ、年間を通して歳時記に彩られている。 日本三大花火大会で有名な「長岡まつり」、武者装束で町を練り歩く「米百俵まつり」など、長岡市の歴史と文化に触れることができる。寺泊温泉は寺泊地区にある2種類の源泉を楽しむことができる温泉地だ。 ひとつは小さな山の地下約1000mほどからガスとともに自噴している約35℃の強食塩泉。もうひとつは地下約2mから湧出している約15℃の炭酸水素塩泉だ。自然に囲まれた小さな温泉宿は海まで車で約5分の好立地。そして肌触りの異なる2つの湯を贅沢に味わいたい。 爽快な風呂上り感の「強食塩温泉」、10種類以上の豊富なイオンの「炭酸水素塩泉」を男湯・女湯でそれぞれ楽しめる（時間帯で入れ替わる）。また日帰り入浴（10:30～15:00）も可能。
