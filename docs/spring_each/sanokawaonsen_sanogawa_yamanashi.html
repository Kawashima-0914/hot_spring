佐野川温泉（さのがわ）

所在地：山梨県

効能ラジウム泉

静岡県と山梨県の県境に位置する佐野川温泉は、山奥にある一軒家度「佐野川温泉旅館」で自噴している。周囲になにもない静かな環境で、昔から地元の人に愛され、素朴なたたずまいを見せる。庭園風露天風呂からは、季節ごとに桜や紅葉の山景色が楽しませてくれる。加水・消毒なしのかけ流しで、内風呂はぬる湯とあつ湯の2つの湯船を備えているから、じっくりと交互浴に勤しもう。自然の沼を利用してニジマスの養殖がおこなわれているため、佐野川温泉旅館でもニジマス料理が供されている。その他にも、山菜などを使った料理や蕎麦など、山あいの温泉旅館ならではのグルメが堪能できる。 単純硫黄温泉(低張性アルカリ性低温泉)ので、地下150メートルから自噴する。無色透明、ゆで卵のような硫黄臭がする。ふわりとした柔らかな湯ざわりで、じっくり入っていると細かな気泡が肌に付着してくる。とくにアトピー性皮膚炎などの皮膚病や冷え性によいとされている。また、飲泉も可能で、糖尿病、痛風、便秘などの緩和が期待できる。
