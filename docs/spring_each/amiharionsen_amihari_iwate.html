網張温泉（あみはり）

所在地：岩手県

泉質単純温泉、硫黄泉
効能にごり、にごり（乳白色）、神経痛、婦人病、高血圧、皮膚病、糖尿病、痔疾

岩手県の休暇村内にある網張温泉は、「大釈の湯」「白泉の湯」「薬師の湯」「仙女の湯」「鹿追足湯」の5カ所の風呂があり、これらを総称して「網張五湯」と呼んでいる。今から約1300年前の飛鳥時代に発見された温泉で、かつては亭沢温泉と呼ばれていた。「網張」の名前は、山岳信仰により一般の入湯が禁止となり周囲に網を張ったからとも、魑魅魍魎を避けるためつる草の網を張ったからとも言われている。「仙女の湯」「鹿追足湯」の2箇所は冬季閉鎖してしまうので、すべての湯を堪能するなら春から秋がおすすめだ。湧水をめぐるウォーキングや葛根田渓谷への送迎など、参加体験イベントも実施しているので、旅の思い出にぜひ参加しておきたい。 単純温泉・硫黄泉（硫化水素型）。元湯は山を登った先の爆裂火口跡にあり、流れ込む沢水を混ぜて温泉が作られて2Kmの引湯管を通ってくる。源泉と沢水が混ざる際の温度によって色が変わるのが特徴で、夏から秋は青みがかった白濁した湯で硫黄臭があり、沢の水が細い冬になると無色透明に近くなる。効能はアトピー性皮膚炎、末梢神経障害、尋常性乾癬など。
