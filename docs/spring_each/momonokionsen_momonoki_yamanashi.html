桃の木温泉（もものき）

所在地：山梨県

効能ラジウム泉

手つかずの自然に包まれる南アルプスの奥地にあることから「秘湯」と呼ばれる桃の木温泉。この湯を楽しめる一軒宿「桃の木温泉さんわそう」は、｢何も足さない、かけ流しの温泉｣をコンセプトに、敷地内で湧出するこの源泉をそのまま入れている。この施設の人気の理由は、自然の中に湧くこの自家噴出温泉と、ペットを同伴する宿泊ができる宿であること。専用ドッグラン付きの部屋や、愛犬と一緒に入れる貸切風呂など、愛犬家にうれしい設備が充実している。食事は、郷土料理やほうとうなべ、馬刺しなどの料理を提供しており、この地の味を堪能できる 神経痛、筋肉痛、関節痛、慢性消化器病、冷え性、疲労回復、健康増進などによいとされるアルカリ性単純温泉。敷地内の源泉から引かれている温泉は、湯量豊富な100％かけ流しの湯。最初から適温で湧出しているので、加温の必要もない。
