ニセコアンヌプリ温泉（にせこあんぬぷり）

所在地：北海道

泉質炭酸水素塩泉、塩化物泉、硫酸塩泉
効能筋肉痛、関節痛、疲労、皮膚病、美肌

ニセコは世界四大スキー場に数えられ、オーストラリアや中国をはじめ海外からのスキー客も多い。その中心がニセコアンヌプリで、標高1302mのニセコ連峰最高峰。いくつものスキー場がつながった規模の大きさと、上質なパウダースノーで人気を博するが、冬以外も楽しい通年型リゾートである。夏・秋は、羊蹄山とニセコ連峰の雄大な山並みを眺めながらのドライブやトレッキングが楽しめる。ラフティングやキャンプなどの幅広いアクティビティが体験できることでもファミリーを中心に人気が高い。一帯は泉質多彩なニセコ温泉郷。「湯めぐり手形」をもって温泉巡りをするのも楽しいだろう。 泉質は塩化物-炭酸水素塩-硫酸塩泉。保温・保湿効果のある塩分を含み、スキーで冷え切った体を温めてくれるまろやかな泉質だ。美肌にいいとされる「メタケイ酸」の含有量も非常に多いのが特徴。重曹（炭酸水素塩）もクレンジング効果から美肌にいいとされており、「美肌の湯」と言えよう。源泉かけ流しの風呂も要されているのもうれしい。
