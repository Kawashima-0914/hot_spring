八ヶ岳南麓温泉（やつがたけなんろく）

所在地：山梨県

八ヶ岳、南アルプス、富士山など美しい山々を望む立地に佇む八ヶ岳南麓温泉。標高約1000mにあることから、満天の星空を堪能できるのも魅力だ。宿泊施設の中には展望台や望遠鏡といった天体観測の設備をそろえているところもあり、夜空の美しさを間近に感じられるだろう。また、キャンプ場やバーベキュー場が併設された施設などもあり、大自然のレジャーを楽しみつつ温泉も堪能できるという贅沢な過ごし方もおすすめ。温泉地のある清里は、牧場やスキー場などの体験型施設や「清里湖大門ダム」「八ヶ岳高原大橋」「谷戸城跡」など景色や歴史を楽しめる施設など、さまざまな観光スポットが充実している。また、夏にはホタル鑑賞ができるスポットもあるため、一年を通じて楽しめる。 ナトリウム-炭酸水素塩泉の泉質で、皮膚の洗浄作用によって、湯上がりには清涼感がある。神経痛、関節痛、冷え性、慢性消化器病、うちみ、くじき、やけど、慢性皮膚病、慢性婦人病など多くの効能が期待できる。飲泉では慢性消化器病・糖尿病・痛風などの適応症があるが、飲用の可否については各施設に確認が必要だ。
