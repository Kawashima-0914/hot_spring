稲子川温泉（いなごかわ）

所在地：静岡県

泉質塩化物泉、硫黄泉
効能神経痛、冷え性、皮膚病、美肌

富士川の支流「稲子川」のほとりにある温泉地で、毎分300Lを超える豊富な湧出量を誇る温泉を、公営の温泉施設「ユー・トリオ」のみで利用している。梅林やバーベキュー場なども整備されているのが特徴の「湯けむりリゾート」だが、最大の特徴は温泉プールがあることだろう。老若男女楽しめることを目指して作られた施設で、温泉だけでなくプールもある。プールで運動と温泉の相乗効果が期待できそうだ。稲子川の上流には天子の七滝と呼ばれる美しい滝がある（現在は台風被害のため入山規制）。貴重な生態系を残した自然豊かな地のめぐみを体験できそうだ。 pH10という強めのアルカリ性。泉質は、硫黄とカルシウムを含む塩化物泉で、保温効果が期待できる。また、硫黄は皮脂を洗い流し、アルカリ性は角質を溶かす一方で、塩分は肌を保湿してくれるので、美肌効果は絶大と言えよう。無色透明ながらほんのりと硫黄の香りが漂う湯。温度が低い源泉をそのままの温度で入れる湯舟もある。
