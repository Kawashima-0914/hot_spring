湯の谷温泉（ゆのたに）

所在地：熊本県

効能硫酸塩泉

熊本県阿蘇郡南阿蘇村にある鳥帽子岳の中腹、標高820mの高原にある阿蘇最高所に湧く温泉地。いにしえの時代より、こんこんと温泉が沸き出ていたことから「湯の谷」と名づけられた。眺望が素晴らしく大観峰や有明海を隔てた島原や雲仙までが一望出来る。昭和天皇も３度宿泊されたという阿蘇観光ホテルは2000年2月に閉鎖。跡地は現在も当時のまま残されており、廃墟となった建物を用いて2005年には映画「輪廻」のロケが行われた。ホテル敷地内には、穴が開いた三角形状の大きな岩「縁結びの石」が置かれており、石の両側から男女が穴の中で固く手を結び合うとその二人は必ず結ばれるという言い伝えがあり、閉館後は阿蘇下田城ふれあい温泉駅の駅舎横に移設された。 古くよりこんこんと湯の谷から湧き出る温泉は、マグネシウム、カルシウム、硫酸塩冷鉱泉。神経痛、筋肉痛、関節痛、五十肩、運動麻痺、関節のこわばり、うちみ、くじき、慢性消化器病、痔疾、冷え性、病後回復期、疲労回復、健康増進、動脈硬化症、きりきず、やけど、慢性皮膚病などに効果があると言われている。
