御殿場乙女温泉乙女2号泉（ごてんばおとめおとめにごうせん）

所在地：静岡県

効能単純温泉

箱根仙石原と御殿場を結び、富士見三峠の一つに数えられる乙女峠の中腹にある。乙女峠は、とめという娘が病身の父を思ってお百度参りの果てに身代わりになったという伝説に基づく名前。山の聖なる力を感じさせる。御殿場の市街地と富士山を一望でき、日中はもちろんのこと、夜は夜景も楽しめる。３つある温泉施設のいずれも露天風呂を持ち、晴れた日は絶景が期待できそうだ。御殿場インターと箱根から近く、観光スポットにも事欠かない。キャンプ場とリゾートホテルがあり、幅広いニーズに応えた滞在ができる。高原にあって夏でも風が涼しく快適に過ごせる。 弱アルカリ性の単純温泉で、浴後はサッパリとした感覚が得られる。源泉温度は低めで、加温して利用されている。美肌効果のあるアルカリ性のお湯の働きもさることながら、富士山を眺めながらゆったりと入浴する開放感によってストレスが解消される効果も大きいだろう。温泉旅行によって、成分による薬理効果だけでなく気分転換による転地効果も得られる。両方の効果を体感できそうな温泉だ。
