志太温泉（しだ）

所在地：静岡県

効能塩化物泉

藤枝の郊外にありながら、地元の人でも知っている人はわずかという町の中の秘湯。現在は「潮生館」一軒のみが営業している。全８室の小さな宿には、浴室が２つも。温泉にこだわるご主人の心意気を感じられる。「割烹旅館」というだけあって、料理も趣向を凝らしたものが味わえる。本館は明治中期、離れの香梅荘も昭和初期に建てられたもので、木造のぬくもりある建物は登録文化財にも指定されている。犬養毅や後藤新平など、数多くの政財界の中心的人物が疲れを癒したという歴史ある温泉地。伝統と職人の心意気を引き継ぎながら、心の底から落ち着ける空間を作り上げている。 昔から存在が知られていたという塩分を含む冷鉱泉を沸かして使っている。泥の層に含まれるガスと塩分が地下水に溶け出し、鉱泉として湧いているのだという。古くから皮膚病への効果があると言われており、明治時代に宿を構えて本格的に温泉地としての歴史を歩み始めた。敷地内で自噴する源泉を、鮮度を損なわぬよう加温してかけ流して使っているのも嬉しい。
