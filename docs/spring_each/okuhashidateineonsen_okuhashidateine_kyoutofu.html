奥橋立伊根温泉（おくはしだていね）

所在地：京都府

効能単純温泉

海岸線のわずかな平地に集落を形成する、京都府下有数の漁業の街、京都府伊根町。過去に放映されたテレビドラマの舞台になったことで全国に知られるようになった漁業の街の東端に湧く温泉が、奥橋立伊根温泉。若狭湾を前に、山の緑を背中に見る温泉は、源泉の湯量が豊富で、施設によってはかけ流しの露天風呂で入ることもできる。温泉の周囲には、約230軒の舟屋が伊根湾を取り囲むように軒を連ねる舟屋群「伊根浦舟屋群」や、日本最古の「浦島太郎伝説」が伝わる風景、衣食住や民俗芸能に関わる文化財などがあり、観光地としても人気が高い。 神経痛や肩こり、筋肉痛や冷え性、疲労回復など多くの効能が期待できる、無色透明の単純温泉。多くは宿泊施設の中にあるが、日帰り温浴が可能な施設もある。閑静な環境にある温泉なので、静養や保養の地としても適している。
