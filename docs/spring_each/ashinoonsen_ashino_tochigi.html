芦野温泉（あしの）

所在地：栃木県

効能単純温泉

那須高原の自然の森と、40面のテニスコートに囲まれた中にある温泉施設「芦野温泉」。「現代の湯治場」と自称する施設内には、2種類の自家源泉を引いた露天風呂をはじめ10種類以上のお風呂があり、施設宿泊者だけでなく日帰り入浴もできる。温泉施設中央部にある「メラの湯」は、美白効果を持つとされる天然温泉に水素を投下した水素温泉。施設の中では、湯治場らしいイベントを季節ごとに開催しており、温泉に入る以外にもさまざまな楽しみ方ができる。旧奥州街道芦野の里にある温泉の周辺は、江戸時代末期まで城下町として栄えた歴史ある場所。散策や観光も楽しめる地域でもある。 泉質は単純温泉で、神経痛や冷え性、筋肉痛や五十肩などへの効能が高いとされている。源泉に薬草を漬け込んて薬効を持たせた「薬草の湯」は、刺激性のある湯で、浸かると肌に痛みを感じるほど。休憩を挟みながら湯治らしいお湯を愉しみたい。
