湯島温泉（ゆしま）

所在地：鹿児島県

効能塩化物泉

薩摩河内市内を流れる川内川の河畔近くに建つ一軒家の中の温泉。効能の高いことから地元の人の間では「漢方の温泉」と呼ばれることもある。温泉地周辺には、大河ドラマのロケ地となった「長野滝」があり、マイナスイオンを放出するパワースポットとしても知られている。戦国時代の資料館である「川内戦国村」も近い。本格的な甲冑を製作・販売する甲冑専門店「甲冑工房　丸武」では、広さ1000坪もの枯山水が魅力の庭園や、映画やドラマなどに使われた甲冑の展示などを見ることができる。ご当地グルメとしては、薩摩川内市の郷土菓子「たまごむっかん」がある。全卵、砂糖、小麦粉、地酒で作られた素朴な蒸し菓子は、湯島温泉のお土産としても良さそう。 湯島温泉の泉質は塩化物泉で、神経痛や慢性消化器病、便秘や婦人病などに効能があるとされている。冷泉を沸かした温泉で飲泉も可能。飲用としては胃腸病への効能が期待できる。
