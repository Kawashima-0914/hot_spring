みやばる温泉（みやばる）

所在地：熊本県

泉質単純温泉、含鉄泉
効能神経痛、リウマチ、腰痛、疲労、ねんざ、冷え性、高血圧、喘息

ゆったりとした風景の中に現れる田園の中の温泉地、宮原温泉。九州自動車道の植木インターから約10分という立地もあって人気が高い。阿蘇の噴煙を遠望する、熊本県らしい景色を楽しむこともできる。熊本駅周辺からも車で行き来しやすいこともあり、熊本城など市街地を観光してから訪れるのもおすすめ。温泉地の周辺では、「釜尾古墳」をはじめ多くの古代遺跡や古墳が発見されている。また、西南戦争の激戦地である「田原坂」や剣豪・宮本武蔵が眠る「武蔵塚公園」など、日本史の舞台になった場所も存在する。宮原温泉のある熊本市北区は農業も盛んで、特にスイカは日本有数の生産量を誇る。宮原温泉に訪れた際は、温泉はもちろん歴史や食事にも注目したい。 宮原温泉では基本的な泉質が、アルカリ性の単純温泉で、肌への刺激が少なく柔らかなお湯が特徴。温泉施設の中には、鉄分を多く含む褐色の湯と無色透明の湯の2種類を楽しむことができる場所もある。褐色のお湯は含鉄泉であり、多くの鉱物イオンと適量のラドンを含む高い効能を持つとされるお湯だ。鎮静造血作用があると言われ、神経痛やリウマチなどへの効果を期待できるほか、美肌効果もありそう。
