原城温泉（はらじょう）

所在地：長崎県

効能炭酸水素塩泉

原城温泉は長崎県南島原市南有馬町丁にある温泉。天草四郎で知られる島原の乱がおこった原城跡のすぐ近くにある。原城温泉がある南有馬には温泉のほかにも、自然、歴史、芸術など様々な楽しみ方ができる。有明海には多数のイルカが生息しており、イルカの声が聞こえるほど近づいて見学ができる。また、近くの原城跡は、キリシタン弾圧の悲惨な歴史を語り継ぐ史跡文化財であり、ツアーに申し込めば専属ガイド付きで歴史を学ぶことができる。南島原市の原城本丸沖合に年に数回、最干潮時に「白洲」といわれる浅瀬が姿を現す。普段は海中に沈んでいて見ることができない。世界中でも珍しく、インド洋、イギリス海岸と原城沖の3ヶ所でしか見ることのできない貴重な場所だ。 温泉の始まりは、大正11年の島原半島南部を中心とした大地震によって湯煙をあげたことだ。原城温泉の泉質はナトリウム炭酸水素塩泉。神経痛、筋肉痛、関節痛、慢性消化器病、痔疾、冷え性、疲労回復、うちみ、くじき、病後回復、健康増進などに効能があるとされる。美肌効果がある泉質で別名「美人の湯」とも呼ばれる。
