猿川温泉（さるかわ）

所在地：群馬県

泉質二酸化炭素泉、塩化物泉
効能にごり、にごり（褐色）、神経痛、リウマチ、筋肉痛、肩こり、胃腸病、冷え性、高血圧、動脈硬化、皮膚病、痔疾

わたらせ渓谷鐵道の水沼駅は、構内に温泉施設「水沼駅温泉センター」がある駅としても知られる。駅舎に沿って造られた細長い施設の浴室と露天風呂は、赤城山麓を源泉とする猿川温泉の湯を引いたもの。「かっぱ風呂」の名称は、駅の近くを流れる渡良瀬川に古くから伝わる「釜が淵の河童伝説」にちなんだもの。大きな窓からは渡良瀬川の対岸や緑を眺望でき、爽やかな気分での入浴ができる。「かっぱ風呂」には食事処や広間なども併設されており、食事処では名物の「きりゅうどん」や「山うなぎ丼」などが食べられる。電車は1時間に1本程度なので、万一乗り過ごしたら温泉へ入って次の電車を待つのも楽しい。 二酸化炭素ナトリウム・カルシウム塩化物・炭酸水素塩冷鉱泉の泉質を持つ猿川温泉は、神経痛や慢性消化器病、冷え性や動脈硬化など多くの効能が期待できる。よく温まり湯冷めしにくい。さらりとした感触とほんのり褐色がかっている湯は泉温が低いため、加温、し循環ろ過装置を使用している。
