新湯田中温泉（しんゆだなか）

所在地：長野県

効能単純温泉

湯田中駅の徒歩数分圏内に温泉街が広がる。夜間瀬川の流れを眼下に見下ろす高台に位置しており、戦後にできた比較的新しい温泉地だ。12軒あった宿は、現在6軒となっており今も残る建物は明治期の独特の作りでタイムスリップしたような気持ちで街並みを楽しめる。駅の反対側にある「湯田中温泉」は俳人・小林一茶との縁が深く、ここの裏山が一茶の俳句創作に利用されたこともあるそう。湯田中の湯に癒されたら、あなたもここで一句詠んでみては。駅に隣接した「楓の館」の投句箱に入れると駅周辺の灯籠に掲げられ一定の期間披露されるかも。 新湯田中温泉には2つの共同浴場がある。透明に澄んだ湯が溢れる「鶴の湯」と、湯の花舞う「亀の湯」だ。お湯自体に主張がないため飽きが来ず長く愛されているのが特徴だ。温泉街は夜間瀬川の流れを眼下に見下ろす高台に位置するため、北信五岳や北アルプスも見える。眺望がよく美人の湯としても知られている。
