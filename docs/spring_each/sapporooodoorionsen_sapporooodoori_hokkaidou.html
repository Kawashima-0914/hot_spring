札幌大通温泉（さっぽろおおどおり）

所在地：北海道

効能塩化物泉

北海道の中心、札幌市。なかでも官庁や企業などが集積し、碁盤の目状の街並が美しい市の中核となるのが中央区。札幌市のシンボルとしてたびたびテレビや映画にも登場する「札幌市時計台」や「大通公園」など観光スポ ットも豊富である。 冬は「さっぽろ雪まつり」、初夏には「YOSAKOIソーラン祭り」など、大きなイベントには全国から多くの観光客が訪れる。昼は「札幌ラーメン」や「ジンギスカン」などのご当地グルメで舌鼓。 夜は日本3大歓楽街のひとつである「すすきの」で遊び、見所は尽きない。歩きまわった旅の疲れを癒すのには温泉が最適。宿泊者専用のこぢんまりとした温泉は筋肉痛や疲労回復に効能がある。交通の利便性も良いので市街地観光の拠点におすすめだ。 宿泊者専用のこぢんまりとした温泉露天風呂。
