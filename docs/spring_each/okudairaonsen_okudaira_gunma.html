奥平温泉（おくだいら）

所在地：群馬県

効能単純温泉

奥平温泉は川手山森林公園の中にある温泉地で、猿ヶ京三国温泉郷のひとつである。奥平温泉の湯を楽しめるのは日帰り入浴施設「遊神館」。広大な敷地内で、大浴場のほか薬湯や露天風呂などさまざまな趣の湯を楽しむことができる。食事処で食べられる打ち立てのそばも人気だ。温泉を堪能した後にぜひ味わいたい。温泉地の周囲はのどかな自然に囲まれており、温泉街のような賑わいはないが、ゆったりと湯浴みを楽しみたい人にはにはうってつけ。気心の知れた友人や家族、恋人と訪れ静かで素朴な時間を過ごしてみてはいかがだろうか。 アルカリ性単純温泉の泉質を持つ奥平温泉は、肌がつるつるすべすべになる美肌の湯としても知られる。そのほかにも神経痛や疲労回復、冷え性や自律神経失調症などさまざまな効能が期待できる。また、遊神館ではゆず湯やリンゴ風呂など四季折々に合わせた薬湯も用意している。こちらも楽しみたいところだ。
