蒲郡温泉（がまごおり）

所在地：愛知県

泉質単純温泉、塩化物泉、ラジウム泉
効能神経痛、筋肉痛、関節痛、疲労、婦人病、冷え性、高血圧、動脈硬化、痔疾

東名高速音羽蒲郡ICから三河湾オレンジロード、国道473号と23号経由で約10㎞ 、JR蒲郡駅から車で5分ほど。蒲郡市は愛知県の中でも有数の温泉観光地で、中でも最大といわれる規模を誇る「蒲郡温泉」は三河湾の名勝・竹島周辺に湧く温泉。 国の天然記念物にも指定されている蒲郡のシンボル・竹島が絵画のような借景となり、古くから皇族や多くの著名な作家が好んで逗留した。春にはつつじ祭り、夏は 納涼船と、四季を通じて風景を楽しむことができる。 「竹島水族館」や「ラグーナテンボス」、「ラグーナフェスティバルマーケット」などレジャー施設も充実。レジャーを楽しんだ後、肌にやさしいと評判のお湯で、ゆっくりと疲れを癒したい。 愛知県で最大といわれる温泉観光地の「蒲郡温泉」。三河湾の名勝・竹島付近のロケーションに湧くだけあり、三河湾の美しい景色を望める眺めの良い湯浴みが楽しめる。無色透明の温泉は、アルカリ性のため、ツルツルして肌にやさしいと評判も高い。神経痛、筋肉痛、関節痛、五十肩、打ち身、慢性消化器病、痔疾、冷え性、疲労回復など 効能は多岐にわたり、湯治に利用する人々も多いとか。三河湾の美しい眺めと良質の温泉が日頃の疲れを癒してくれる。
