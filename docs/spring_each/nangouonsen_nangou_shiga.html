南郷温泉（なんごう）

所在地：滋賀県

効能ラジウム泉

古くから山紫水明の都といわれている滋賀県大津市は、比叡の山並をはるかに拝む音羽山系岩間山の麓に広がる地。京都府と滋賀県の境に位置する岩間山の山中からこんこんと湧き出ている地下水を用いているのが南郷温泉 。 京滋バイパス石山ICより約10分、琵琶湖から流れる清流瀬田川の畔に佇む料理旅館「二葉屋」が一軒宿で、瀬田蜆や二ゴロブナなど琵琶湖の豊かな恵みが楽しめる。特に琵琶湖の鯉の姿煮は食通も唸らせるほどの美 味さだとか。 湖国会席を堪能したら、展望風呂で瀬田川の美しい景色を愛でながら身も心も芯まで温めて、しっとりした湯浴みの時間を過ごしたい。特に、夕暮れ時の光景は美しい絵画を見ているかのようで、趣きある格別な風景が楽しめる。 加温・循環式。
