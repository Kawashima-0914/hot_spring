飯森山温泉（いいもりやま）

所在地：山形県

効能単純温泉

市街地からも近い複合型施設「かんぽの宿酒田」は、鳥海山を一望できる高台に位置している。飯森山温泉を源泉とした、露天風呂をはじめ歩行湯や寝湯など12種類もの風呂が楽しめ、日帰り入浴・宿泊どちらも対応している。屋内外にはテニスコート、ゲートボール場といったスポーツ施設も整備されているので、汗を流した後に温泉に浸かればすっきりとした気持ちになれそう。周囲には酒田市美術館、土門拳記念館、出羽遊心館などの観光施設も多いため、昼間に観光施設を巡って、夜は温泉で疲れを癒やすコースはいかがだろうか。 神経痛、筋肉痛、関節痛、五十肩、うつ状態などに効能があるとされる単純温泉（低張性アルカリ性低温泉）。無色透明、無臭の湯はクセがなく、どんな人でも入りやすいのが特徴だ。湯量が豊富で贅沢な気分で温泉に浸かることができる。
