西山温泉（にしやま）

所在地：山梨県

効能筋肉痛、胃腸病、便秘、美肌

ＪＲ「身延駅」から山梨交通バス奈良田行きで約90分、南アルプスの最高峰・北岳を源流とする早川沿いに湧く山間の温泉地。早川沿いに延びる南アルプス街道をさかのぼって温泉地へと向かう道中、目を楽しませてくれるのは早川の素晴らしい渓谷美。川幅の狭い所では巨岩が味わい深い自然の造形美を見せ、広いところでは水面が太陽の光を浴びてきらきらと輝く様が目を奪う。そんな自然の芸術を愛でながらたどり着いた先の温泉地は、山間の奥まったロケーションに佇み、歴史ある秘湯の風情たっぷり。徳川家康や武田信玄など、名だたる戦国武将も湯浴みしたといわれる古来から名高い温泉である。 毎分1600リットルの湯が自噴する、ナトリウム・カルシウム－硫酸塩・塩化物泉（低張性アルカリ性高温泉）。胃腸病や筋肉痛、便秘、美肌などへの効能があるとされる。無色透明でわずかに温泉特有の硫黄臭がする湯が特徴。
