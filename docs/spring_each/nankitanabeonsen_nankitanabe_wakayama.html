南紀田辺温泉（なんきたなべ）

所在地：和歌山県

効能塩化物泉

どこまでも続く青い海、太陽に煌めく白い砂。南紀白浜は和歌山県屈指のリゾート地だ。海岸線には海水浴場や景勝地がどこまでも連なり、南紀白浜の魅力を余すところなく満喫できる。 南紀白浜には温泉地がたくさんあり、温泉郷をはじめ一軒宿を含めるとその数はかなりのもの。南紀田辺温泉は内ノ浦湾を望む新庄町に湧く一軒宿で、白浜からは車で約10分とアクセスも良好。 太平洋を一望できる温泉大浴場、潮風を感じながら浸かる野趣豊かな露天風呂、室内プール、室外プール、テニスコート、パターゴルフなどレジャー施設も多彩。潮騒を感じながら思い切り身体を動かしたら、良質な天然温泉でゆったりと湯浴みを楽しみ、一日の疲れも吹き飛ぶリラックスタイムを過ごそう。 pH8.3、トロトロの肌触りが嬉しい美肌のお湯。オーシャンビューの露天風呂は絶景だ。
