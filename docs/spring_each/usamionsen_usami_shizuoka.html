宇佐美温泉（うさみ）

所在地：静岡県

効能塩化物泉

JR伊東線 宇佐美駅からほど近い場所にある温泉地。熱海と伊東に挟まれた場所にあり、毎年海水浴シーズンになると多くの親子連れでにぎわいを見せる。海水浴場は1km以上続く遠浅の広々した砂浜で、相模湾が目の前に広がっている。近くには、永久の世界平和を祈る姿を模しているという「宇佐美観音」も。高さが５０メートルの座像で日本一の大観音様であり、見ごたえは抜群。泉質は塩化物泉で、「温まりの湯」と呼ばれており、海に近い宇佐美の温泉は湯冷めしにくく疲労回復や美肌効果なども期待できる。また、イチゴやミカン狩りも楽しむことができるのが魅力的だ。 泉質は塩化物泉で、弱アルカリ性。「温まりの湯」と呼ばれており、海に近い宇佐美の温泉は保温効果のある塩分を含むため湯冷めしにくいと言われている。サラッとしたやわらかい肌触りも特徴だ。適応症は、神経痛、筋肉痛、間接炎、や五十肩、特に婦人病の諸失患に効果がある。また、疲労回復や美肌効果なども期待できる。
