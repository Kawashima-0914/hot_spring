三富温泉郷（みとみ）

所在地：山梨県

泉質単純温泉、塩化物泉、硫酸塩泉
効能神経痛、筋肉痛、肩こり、疲労、冷え性

三富温泉郷は笛吹川上流の三富地区に湧く「一の橋温泉」「川浦温泉」「三富温泉」の総称である。日帰り温泉を含め4軒の入浴施設が点在し、趣の違う温泉が楽しめる。国内屈指の景勝地「西沢渓谷」にも近く、ハイキングやトレッキング、また療養としても最適な温泉地だ。周囲には国宝に指定されている「清白寺」や伝統ある神社「大井俣窪八幡神社」など、重要文化財や史跡も多い。これらを巡る温泉旅を計画するのもおすすめだ。さらに山梨といえば郷土料理「ほうとう」や桃、ぶどう、さくらんぼなどのグルメが外せない。ほうとうを提供する食事処は多いので、食べ比べをしてみるのもおすすめだ。 施設によって引かれている温泉が違うため、いくつかの施設に立ち寄ってみるのもおすすめだ。神経痛、筋肉痛、冷え性、疲労などによいとされるアルカリ性単純温泉と、きりきず、皮膚病、婦人病、動脈硬化、便秘などによいとされるナトリウム・カルシウム−硫酸塩・塩化物泉がある。
