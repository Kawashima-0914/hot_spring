河津浜温泉（かわづはま）

所在地：静岡県

効能塩化物泉

伊豆急行「河津駅」のほど近く、河津浜に面した温泉地。駅から徒歩5分ほどのところに河津浜海水浴場があり、夏には海水浴を楽しめる。隣接する今井浜海水浴場に比べると混雑は少なく、穴場的なスポットだ。 また、全国的に町の名物として知られるのが、“本州一の早咲き桜”として名を馳せ、河津川沿いに連なる河津桜。毎年、2月上旬から「河津桜まつり」が開催され、多くの観光客で賑わう。 この他にも、「かわづ花菖蒲園」「かわづカーネーション見本園」「河津バガテル公園」など花の名所が目白押し。それぞれ趣きの異なった花の美しさを堪能できる。青く輝く海と、色とりどりの花々、そして温泉。ごく自然に心が和む、穏やかな空気に包まれた海辺の温泉地だ。 7つの温泉からなる河津温泉郷のひとつで、河津川の河口を中心とした河津浜海岸周辺の温泉。塩分を含んだ湯は、体を芯から温め湯冷めしにくい。神経痛や婦人病のほか、外傷にも効くといわれている。小中規模の湯宿の他、国道135号沿いの浜辺に素朴な共同浴場があり、無料で入浴が楽しめる。
