神山温泉（かみやま）

所在地：徳島県

泉質炭酸水素塩泉、塩化物泉
効能神経痛、リウマチ、筋肉痛、関節痛、冷え性、皮膚病

徳島県神山町にある温泉。徳島市内から車で約35分という手軽な距離にありながら、街中とはまったく異なる里山環境の中にある。西日本でも屈指の名湯を誇る佇まいで神山産の材木をふんだんに使用し、落ち着いた里山ならではの香り。バーベキューなどもできるキャンプ場や、道の駅の「旬の市場神山」や「ふれあい市」特産品の販売も盛んで、里山を満喫する為の施設が揃っている。農村ふれあい公園は、のどかな雰囲気で、春は菜の花・しだれ桜・藤などが咲く。近くには30メートルの落差を持つ「神通滝」がある。冬には氷瀑となり神々しい姿を見せる。 泉質はナトリウム塩化物・炭酸水素泉。ややぬめりを含み、ゆっくり浸かるととても肌がつるつるして身も心もほかほかになる。、四国八十八ヶ所11番札所と12番札所の中間に位置し、お遍路の旅の疲れをいやすのにも最適だ。
