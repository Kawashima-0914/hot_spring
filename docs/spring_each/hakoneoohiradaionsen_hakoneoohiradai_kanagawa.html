箱根大平台温泉（はこねおおひらだい）

所在地：神奈川県

効能塩化物泉

戦国時代からの歴史ある場所で、箱根十七湯の一つと知られている。芦ノ湖では、逆さ富士を見られることもある。「かながわの景勝50選」に選ばれている大涌谷から眺める富士山や、運が良ければ逆さ富士が見られる芦ノ湖など、周辺では富士山の絶景が楽しめる。さらに旧跡も多く、「林泉寺」や「山神神社」などの神社やお寺を巡ることもできる。「蛙の滝」「姫の水」「仙元の泉」など伝承のある場所も多く、きれいな水と山の緑が楽しめそうだ。大平台そのものが、村人が大蛇を退治しようとしたときに大蛇がのたうち回って作ったという伝説がある。観光をして動き疲れたら、温泉に入って、汗を流そう。きっとリラックスできるだろう。 大平台温泉郷は戦国時代からの古い村落でしたが、昭和24年地元の有志たちによって、温泉が掘り当てられ、箱根十七湯に数えられるようになりました。湧き水や滝など水が自慢の観光スポットがあるほど水がきれいな地域で、そんな場所で湧き出た温泉を楽しむことができます。温泉は無色透明で、泉質はナトリウム塩化物泉になります。
