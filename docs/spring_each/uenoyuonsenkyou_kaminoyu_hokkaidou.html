上の湯温泉郷（かみのゆ）

所在地：北海道

泉質炭酸水素塩泉、塩化物泉
効能美肌、神経痛、筋肉痛、婦人病、胃腸病、冷え性、皮膚病、便秘

落部川上流の山あいに位置する、全国でも珍しい高温の自噴泉である上の湯温泉郷。その昔、アイヌの人々が狩猟の疲れを癒やすため、また療養のために常浴していたと伝えられている。入浴施設は「銀婚湯」と「パシフィック温泉ホテル　清龍園」の2軒あり、どちらも日帰り入浴が可能。周囲には目立つ観光スポットはないが、豊かな自然に囲まれた閑静な場所なので、療養に専念した人にはぴったりだろう。車での移動ができるのであれば、噴火湾を一望できる「噴火湾パノラマパーク」や眼下に広がる青々とした海を望む「立待岬」などに足を伸ばしてみるとよいだろう。 弱アルカリ性のナトリウムー塩化物・炭酸水素塩泉で、飲泉も可能。 神経痛、筋肉痛、関節痛、ちみ、くじき、痔疾、慢性消化器病、慢性皮膚病、慢性婦人病、冷え性、きりきず、やけどなどのほか、飲用としては慢性消化器病や便秘に効果があるとされている。とくに皮膚病にはよく効くといわれ、やけどがキレイになると評判だ。
