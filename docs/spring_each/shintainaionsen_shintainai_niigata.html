新胎内温泉（しんたいない）

所在地：新潟県

泉質炭酸水素塩泉、硫酸塩泉
効能美肌、神経痛、筋肉痛、冷え性、高血圧、動脈硬化、皮膚病、美肌

飯豊連峰を源とする「胎内川」、そして、緑豊かなブナ林が広がる自然豊かな環境にある新胎内温泉。この地に佇む高原リゾート「ロイヤル胎内パークホテル」が一軒宿だ。展望露天風呂やサウナなども充実した宿で、高級感のある建物が優雅な気持ちにさせてくれそう。敷地内にはバーベキューガーデンやテニス場、スワンボートなどもあり、敷地外に出ることなくレジャーを楽しめる。また、車で5分程度の場所には、「胎内スキー場」や「胎内フラワーパーク」、「スポーツハウスキャンプ場」など多くのレジャー施設があり、温泉と併せて楽しめるリフレッシュに好適な施設となっている。 ナトリウムを含む、ナトリウム－炭酸水素塩・硫酸塩温泉で、動脈硬化症、きりきず、やけど、慢性皮膚病などによいとされ、とくに高血圧症の血圧降下作用、神経痛にもよく効くと言われている。また、皮膚の角質を軟化させ皮脂をさっぱりさせる。美肌効果が期待できる「美人の湯」でもあり、ぬるっとした濃厚な湯ざわりが特徴。
