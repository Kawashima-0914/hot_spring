幡谷温泉（はたや）

所在地：群馬県

効能神経痛、筋肉痛、肩こり、関節痛、ねんざ、打撲、痔疾

武尊山麓、国道120号線から花咲温泉に向かう途中にある温泉地。猟師だった先代のケガから発見に至った場所で、地元で評判の比較的新しい温泉地である。温泉地から車で40分ほどの場所には、日本最大級の高層湿原地帯である尾瀬ヶ原がある。そのほか、15km圏内に日光白根山ロープウェー、丸沼、菅沼と、自然の景観が美しいスポットがそろう。また、ウォータースポーツを体験したいのなら、プロのガイドが案内する「フィールドアース尾瀬」へ。天然スライダーや天然プール、またキャニオニングや水上ラフティングなども楽しめ、特に夏場に人気のスポットだ。水遊びで疲れた体を、幡谷温泉で温め癒やす旅行プランはいかがだろうか。 泉質は匂いが少なく無色透明、一般的に入りやすい単純温泉。神経痛、筋肉痛、関節痛、五十肩、運動まひ、くじき、打ち身など幅広い効能を持つ。その確かな効能で、地元の人に愛されている。入浴後はしっとりとスベスベした肌になるとして、女性にも人気。
