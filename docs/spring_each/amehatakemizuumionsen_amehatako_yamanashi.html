雨畑湖温泉（あめはたこ）

所在地：山梨県

雨畑湖の目の前、療養や休養にも最適な人里離れた閑静な温泉地・雨畑湖温泉。「すず里の湯」のみで、その湯を堪能できる。小・中学校の跡地を利用して建てられた施設で、日帰り入浴はもちろん、隣接するヴィラ雨畑に宿泊も可能。ヴィラの宿泊客の入浴は無料だ。施設にはバーベキュー場や食事処、そして体育館の貸し出しと、小・中学校跡地ならではの活用法もある。また、南アルプスへのハイキングや早川渓谷を巡る自然探訪の拠点としてもぴったり。秋には見事な紅葉が楽しめる。さらに「早川町ジビエシカ肉」や「生芋こんにゃく」「山ぶどうワイン」「南アルプスハム」など、多彩な特産品を求めてグルメ巡りをしてみるのもおすすめだ。 泉質は単純硫黄冷鉱泉（低張性中性冷鉱泉）で、豊富な温泉成分を含む「療養泉」と認められている。神経痛、筋肉痛、運動麻痺、うちみ、くじき、慢性消化器病、痔疾、冷え性、慢性皮膚病、慢性婦人病、糖尿病など効能は多岐にわたる。特に肌への効能が高いとされ、慢性皮膚病やきりきず、やけどによい湯として知られている。
