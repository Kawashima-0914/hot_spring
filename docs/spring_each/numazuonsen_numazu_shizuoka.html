沼津温泉（ぬまづ）

所在地：静岡県

効能塩化物泉

東名「沼津IC」から車で約15分の場所にある「ニューウェルサンピア沼津」に湧く温泉。アクセス良好でビジネス・観光の拠点にも便利だ。沼津市内では珍しい天然温泉。静岡県には「県民の健康を増進し、世界の健康福祉に 貢献する」を目的とした「ファルマバレープロジェクト」がある。 そこには、お湯へのこだわりともてなしを評価する「かかりつけ湯」という制度があり、温泉の衛生管理や食事やホスピタリティのすべてを兼ね備えている 宿が登録されている。 この「沼津 天然温泉」は「かかりつけ湯」として選定されており、信頼性の高い温泉として多くの人に愛用されてきた。富士山のお膝元でたっぷりとしたお湯に浸かり、日本ならではの休息日を堪能したい。 無色透明で強い塩味、苦味があり、湯冷めしにくい。
