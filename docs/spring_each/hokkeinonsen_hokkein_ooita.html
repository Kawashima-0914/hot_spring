法華院温泉（ほっけいん）

所在地：大分県

効能硫酸塩泉

九重連山の中央にある湿地帯、坊ガツルの北西稜にある温泉地、法華院温泉。登山道の中にある温泉で、標高1,303メートルの場所にある「九州最高所の温泉」でもある。九重連山の登山基地としても最適な温泉地で、徒歩でのみ入れる場所にあるため、入る人は登山者にほぼ限定される。温泉地を含む町内の温泉は「久住高原温泉郷」と総称され、竹田市の他の地域の長湯温泉及び竹田温泉、荻温泉とともに「竹田温泉群」として国民保養温泉地に指定されている。温泉地の眼下に広がる坊ガツルは、ラムサール条約湿地にも登録され、春頃には天然記念物であるミヤマキリシマの咲き乱れる景色を臨むこともできる。 加水・循環なしの源泉かけ流し硫酸塩泉で、お湯は無色透明でほのかな硫黄臭がするのが特徴。神経痛や筋肉痛、打ち身やくじき、また冷え性や疲労回復への効能も期待できる。自分の足でのみ辿り着くことができる温泉は、たどり着いた喜びも相まって格別な癒やしを与えてくれそうだ。
