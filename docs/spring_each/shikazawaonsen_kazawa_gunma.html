鹿沢温泉（かざわ）

所在地：群馬県

効能炭酸水素塩泉

リゾート地である嬬恋村を長野県側から訪ねると、地蔵峠を越えて進む県道に100体の観音像が並ぶ。「湯道」とも呼ばれるこの県道の、100体目の観音像の奥が鹿沢温泉。千年以上の歴史があるとされる温泉で、江戸時代に鹿が入湯して傷を癒やしているところを発見したことから「鹿沢」の名が付いたとされる。戦前からスキー場として賑わっていた温泉地は、「雪よ岩よ我らが宿り」で始まる唱歌『雪山賛歌』の生まれた地。現在は石碑が建てられている。温泉と深い雪と自然に恵まれる嬬恋村は沢の水がたまだれのように流れ落ちる「たまだれの滝」や、湯の丸山の麓から頂上にかけて60万株にも及ぶレンゲツツジの群生地など、美しい景色に出会える観光地でもある。 マグネシウム・ナトリウム-炭酸水素塩泉という泉質を持ち、神経痛や筋肉痛、運動マヒやくじき、やけどや胃腸病、婦人病など多くの効能が期待できる。古くから湯治場として親しまれた温泉地でもあり、療養として訪れる人も少なくない。
