倶知安温泉（くっちゃん）

所在地：北海道

泉質炭酸水素塩泉、塩化物泉
効能美肌、神経痛、筋肉痛、関節痛、疲労、冷え性、痔疾

そびえ立つ羊蹄山が絶好のロケーションで眺められる、開放感に溢れている温泉地だ。”遅い春を告げる倶知安町の名所”として近年人気急増している「三島さんちの芝桜」は倶知安町在住の三島さんがご好意で一般開放しているご自宅の芝ざくら庭園だ。この芝桜庭園は2000年代初めころ、三島さんが農家を引退後に芝桜を植えたことが始まり。残雪の羊蹄山とニセコ連峰に見立てて植えられたそうで、今や満開時には約４千平方メートルの花壇に、色とりどりの花々が鮮やかに咲き乱れる。見頃は5月下旬～6月中旬で入場料は無料。倶知安町に行ったら三島さんが手間隙かけて育て上げた芝ざくらをぜひ堪能したい。 泉質はナトリウム-炭酸水素泉・塩化物温泉。炭酸水素泉は、肌の不要な角質や毛穴の汚れを除く効果があるため、湯あがりは肌がすべすべになるといわれており「美肌の湯」とも呼ばれているため、特に女性客に人気が高い。「ホテルようてい」の露天風呂からは蝦夷富士と呼ばれる羊蹄山の四季折々の姿を存分に楽しめる。
