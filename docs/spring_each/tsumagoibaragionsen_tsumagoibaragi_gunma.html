嬬恋バラギ温泉（つまごいばらぎ）

所在地：群馬県

効能単純温泉

標高700m～1400m、冷涼で爽やかな高原地帯に位置する嬬恋村。「日本百名山」の浅間山、四阿山、草津白根山を望むバラギ湖があり、嬬恋バラギ温泉はその湖畔にある温泉地だ。森に囲まれ、鳥のさえずりに耳を傾けながらの入浴は、体の内側まで洗い流してくれるよう。周囲には「無印良品カンパーニャ嬬恋キャンプ場」があり、釣り体験やカヤック、さらにバードウォッチングといったアウトドア教室が充実。バラギ湖はフライ・ルアー、ボート釣りなどができる釣り好きにはたまらないスポット。冬には氷上でのワカサギ釣りも楽しめる。 アルカリ性単純温泉の泉質を持つ嬬恋バラギ温泉は、無色透明、無味無臭の湯が特徴で、湯のあたりが柔らかい。神経痛、関節痛、慢性消化器病などにも効果的とされている。疲労回復にも効果が期待できるため、周囲のスキー場などで1日遊んで疲れた体を癒やすのにも最適。
