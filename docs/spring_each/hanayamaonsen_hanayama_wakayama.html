花山温泉（はなやま）

所在地：和歌山県

効能塩化物泉

花山温泉は和歌山県和歌山市鳴神にある温泉。炭酸ガスの圧力のみで噴出している。湯の成分には1,742.4mgもの炭酸ガスが含まれている貴重な温泉。他にはナトリウム・カルシウム・マグネシウム・鉄分が多く含まれており、神経痛・慢性消化器病・不眠症・高血圧症・動脈硬化症など、20種以上に及ぶ効能が期待できる。湯船のふちに堆積した温泉成分の結晶が、本物の天然温泉であることの証。このエリアにある「和歌浦天満宮」は学問の神としてまつられており、毎年多くの受験生が参拝に訪れている。 花山温泉の泉質は二酸化炭素・鉄・カルシウム・マグネシウム・塩化物・炭酸水素塩泉で構成されている。湧き出した際には無色透明だったお湯が、空気に触れると茶褐色へと変色する。赤色を発しているのは鉄で、湯を結晶化させるのは炭酸カルシウム、そして湯の花として水面に浮かんでいるのはカルシウムによるもの。
