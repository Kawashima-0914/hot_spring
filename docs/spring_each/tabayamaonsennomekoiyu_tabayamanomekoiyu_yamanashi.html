丹波山温泉のめこい湯（たばやまのめこいゆ）

所在地：山梨県

丹波山村の方言に「のめっこい」という言葉がある。「つるつる、すべすべ」という意味の言葉で、「丹波山温泉　のめこい湯」の「のめこい」は、この「のめっこい」からとったもの。その名と意味の通り「丹波山温泉　のめこい湯」は、浸かると肌がつるつる、すべすべする美肌の湯として知られる、この「のめっこい」湯を楽しめる唯一の施設。多摩川源流に湧く名湯で、施設では露天風呂のほか、男女入れ替え制の「ローマ風呂」や「和風風呂」などユニークな工夫で温泉を楽しめるようにしている。露天風呂に入ればお湯に浸かりながら、丹波山から吹く風も丹波山川のせせらぎの音も感じられる。 単純硫黄温泉（アルカリ性低張性高温泉）の泉質で、とくにアトピー性皮膚炎や慢性湿疹、尋常性乾癬、皮膚化膿症など、肌によいとされている。また、湯上がりは肌がつるつるすべすべになる美肌の湯としても知られる。他にも神経痛や筋肉痛、冷え性、疲労回復など幅広い効能が期待できる。
