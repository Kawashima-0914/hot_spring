下仁田温泉（しもにた）

所在地：群馬県

効能炭酸水素塩泉

きれいな水に恵まれた環境を活かして生産するコンニャクイモで有名な下仁田町。鏑川の支流のほとりには、ただ一軒の宿が運営する下仁田温泉がある。川辺で傷ついたイノシシが湯治しているところを村人が発見したことに始まった温泉で、湯治場として親しまれてきた。2万平方メートルもの広大な敷地を持つ宿は、温泉とともに、庭園や猪鹿雉牧場、農園や自然遊歩道などがある。四季折々の美しい風景に囲まれて入る温泉は、体だけではなく心の療養にもよさそう。宿では自家栽培の野菜や、周囲の山で収穫された山菜やキノコ、食聖・北大路魯山人が絶賛したという猪肉を楽しむこともできる。 日本でも数少ない含二酸化炭素・炭酸水素塩泉という泉質を持つ下仁田温泉。毛細血管を広げて血圧を下げる効果を期待できる。中でも胃腸の働きを整える効果があるとされ、肌はなめらかな手触りに。その他にも、消化器病、きりきず、やけど、心臓病、糖尿病、高血圧などに効能があるとされている。
