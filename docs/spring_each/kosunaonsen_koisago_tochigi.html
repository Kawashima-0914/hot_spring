小砂温泉（こいさご）

所在地：栃木県

効能単純温泉

「日本で最も美しい村連合」に加盟する山村・小砂地区にある温泉地。奈良時代から続く「小砂焼」は、「金結晶」と呼ばれる独自の風合いが特徴。陶器づくりの伝統と豊かな山林を活かした環境に配慮した「芸術の森」づくりも行われている。ほかにも、トレラン大会や農村体験などのイベントも精力的に行っており、早くから都市住民との関係を築いてきた。小砂温泉が属する馬頭温泉郷は5つの小さな温泉地の総称であり、温泉水を使ったフグの養殖はそのうちの1つで行われている。しかし、もちろん小砂でも「温泉トラフグ」の料理を味わえる。まろやかな地下水で炊き上げた地元の米とともに味わえば、都会での疲れを癒してくれるだろう。 泉質はアルカリ性単純泉。地下1,200mから湧き出す温泉は、湯量豊富で適温のため、加水も加温もせず、源泉をそのままにかけ流している。新鮮な湯を引くのは「美玉の湯」一軒のみ。贅沢な湯使いによって、地元の人から遠方の温泉ファンまで幅広く愛されている。微量のラドンも含み、健康効果が期待できる。
